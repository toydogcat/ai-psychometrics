## 詳細解析

### 單選題
1. **(B)** $f'(a)$ 是 $x=a$ 處的導數，即切線斜率。
2. **(C)** 定義 $dy = f'(x) dx = 2x dx$。
3. **(B)** 連鎖律：$\frac{d}{dt}(\frac{4}{3}\pi r^3) = \frac{4}{3}\pi (3r^2) \frac{dr}{dt} = 4\pi r^2 \frac{dr}{dt}$。
4. **(B)** $a=3$，$f(3)=9$，$f'(3)=6$。$L(3.01) = 9 + 6(0.01) = 9.06$。
5. **(B)** $\Delta y$ 實際變化，近似於切線上的變化 $dy$。
6. **(C)** 所有變數都是時間 $t$ 的函數，必須對 $t$ 微分。
7. **(B)** $\frac{dA}{dt} = 2x \frac{dx}{dt} = 2(5)(2) = 20$。
8. **(B)** $A = \pi r^2 \implies dA = 2\pi r dr \implies \frac{dA}{A} = \frac{2\pi r dr}{\pi r^2} = 2\frac{dr}{r}$。
9. **(C)** $dy = f'(x)dx \implies f'(x) = \cos x \implies f(x) = \sin x$。
10. **(A)** $f(1)=1, f'(1)=0.5$。$L(1.02) = 1 + 0.5(0.02) = 1.01$。

### 多選題
11. **(A, B, C, D)** 全對。線性逼近即為使用切線，越遠離切點誤差越大。
12. **(A, B, C, D)** 這是微分量的嚴格幾何定義。
13. **(A, C, D)** (B) 錯誤，必須「先微分，再代入特定數值」，否則微出來全變 0。
14. **(A, B, C)** 梯長常數，腳外滑($dx/dt>0$)則頂下降($dy/dt<0$)，高度減少。
15. **(A, B)** 誤差取決於步長 $\Delta x$ 以及函數彎曲程度（二階導數 $f''$ 代表偏離直線的程度）。
16. **(A, B, C)** 利用指數律的對數微分，這些變數的相對誤差倍數就是其指數次方。(D) 錯誤，相對誤差是 $dy/y$。
17. **(A, B, D)** 畢氏定理，對 $t$ 隱微分。若 $x,y$ 增，等式右邊也增，所以 $z$ 增。
18. **(A, C)** 切線極靠近曲線時（$dx$ 小，或曲線平直），逼近最準。
19. **(A, B, C, D)** $e^0=1$，導為 1。$L(x) = 1+x$。代入得 $1.1$。全對。
20. **(A, B, C, D)** $\sin x \approx x$ 在 0 附近非常常用，全對。

### 填充題
21. **$f(a) + f'(a)(x-a)$**。
22. **$4x^3 dx$**。
23. **$2\pi r \frac{dr}{dt}$**。
24. **$1.02$**：$1 + 0.02 = 1.02$。
25. **$3\%$**：$\frac{dV}{V} = 3\frac{dx}{x}$。
26. **$-2$**：$x(2) + x(dy/dt) = 0 \implies dy/dt = -2$。
27. **$\cos x \, dx$**。
28. **$x$**：因 $L(x) = \sin 0 + \cos 0(x-0) = x$。
29. **$\Delta x = dx$**：微分量定義中自變數的變化。
30. **$2x\frac{dx}{dt} + 2y\frac{dy}{dt}$**：對 $x^2+y^2$ 取時間的連鎖律。
