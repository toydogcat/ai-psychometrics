# 課程：微積分上 - 第 18 週 - 期末綜合考試 (Final Exam)

本文件為「微積分上」課程的期末綜合考試題庫，涵蓋從第 1 週到第 17 週的所有核心知識點。

---

## 一、 複習大綱 (Review Summary)

### 1. 極限與連續 (W1-W4)
*   極限的定義、左/右極限、無窮極限。
*   連續性的判別、中間值定理 (IVT)。
*   重要極限：$\lim_{x\to 0} \frac{\sin x}{x} = 1$。

### 2. 導數與微分技巧 (W5-W8)
*   導數定義（切線斜率、瞬時變化率）。
*   基本規則：冪次、乘積、商數、連鎖律。
*   隱函數微分、相關變率 (Related Rates)。

### 3. 導數的應用 (W9-W12)
*   極值定理、均值定理 (MVT)。
*   函數圖形判定：遞增/遞減、凹凸性、反曲點。
*   羅必達法則 (L'Hôpital's Rule)、最佳化問題。

### 4. 積分學基礎 (W13-W15)
*   反導數與不定積分。
*   黎曼和與定積分定義。
*   微積分基本定理 (FTC) Part 1 & 2。

### 5. 進階積分與應用 (W16-W17)
*   代換積分法 (Substitution Rule)。
*   曲線面積、旋轉體體積 (Disk, Washer, Shell Methods)。

## 二、 期末考試題庫 (Final Exam Item Bank)

### 1. 單選題 (Single Choice) - 共 20 題

1. **Q1**: $\lim_{x \to 2} \frac{x^2 - 4}{x - 2} = $ ？
   - (A) 0 (B) 2 (C) 4 (D) 不存在
   - *答案*: C (W1)

2. **Q2**: 下列哪種情況會導致函數在 $x=c$ 不連續？
   - (A) $f(c)$ 定義存在 (B) $\lim_{x\to c} f(x)$ 存在 (C) $\lim_{x\to c} f(x) = f(c)$ (D) $\lim_{x\to c^-} f(x) \neq \lim_{x\to c^+} f(x)$
   - *答案*: D (W2)

3. **Q3**: 若 $f(x) = \sin(x^2)$，則 $f'(x) = $ ？
   - (A) $\cos(x^2)$ (B) $2x \cos(x^2)$ (C) $-2x \cos(x^2)$ (D) $2 \sin x \cos x$
   - *答案*: B (W6)

4. **Q4**: 隱函數 $x^2 + y^2 = 25$ 在點 $(3, 4)$ 的切線斜率為？
   - (A) -3/4 (B) 3/4 (C) -4/3 (D) 4/3
   - *答案*: A (W7)

5. **Q5**: 根據均值定理 (MVT)，若 $f$ 在 $[1, 3]$ 上可微，則存在 $c \in (1, 3)$ 使得 $f'(c) = $ ？
   - (A) $f(3)-f(1)$ (B) $\frac{f(3)-f(1)}{2}$ (C) 0 (D) $f(2)$
   - *答案*: B (W9)

6. **Q6**: 若 $f''(x) > 0$，則函數圖形為？
   - (A) 遞增 (B) 遞減 (C) 向上凹 (Concave Up) (D) 向下凹
   - *答案*: C (W10)

7. **Q7**: $\lim_{x \to 0} \frac{e^x - 1}{x} = $ ？
   - (A) 0 (B) 1 (C) $e$ (D) $\infty$
   - *答案*: B (W11)

8. **Q8**: $\int (3x^2 + 2x) dx = $ ？
   - (A) $x^3 + x^2 + C$ (B) $6x + 2 + C$ (C) $x^3 + 2x^2 + C$ (D) $3x^3 + x^2 + C$
   - *答案*: A (W13)

9. **Q9**: $\frac{d}{dx} \int_1^x \sqrt{t^3 + 1} dt = $ ？
   - (A) $\sqrt{x^3 + 1}$ (B) $\frac{2}{3}(x^3+1)^{3/2}$ (C) $\sqrt{t^3 + 1}$ (D) 0
   - *答案*: A (W15)

10. **Q10**: $\int_0^1 e^{2x} dx = $ ？
    - (A) $e^2 - 1$ (B) $\frac{1}{2}(e^2 - 1)$ (C) $2(e^2 - 1)$ (D) $e^2$
    - *答案*: B (W16)

11. **Q11**: 求 $y=x^2$ 與 $y=x$ 在 $[0, 1]$ 圍成的面積？
    - (A) 1/2 (B) 1/3 (C) 1/6 (D) 1/12
    - *答案*: C (W17)

12. **Q12**: 繞 x 軸旋轉 $y=x$ 在 $[0, 1]$ 的體積（圓錐體）為？
    - (A) $\pi/2$ (B) $\pi/3$ (C) $\pi/4$ (D) $\pi$
    - *答案*: B (W17)

13. **Q13**: 函數 $f(x) = |x|$ 在 $x=0$ 處？
    - (A) 連續且可微 (B) 連續但不可微 (C) 不連續但可微 (D) 以上皆非
    - *答案*: B (W5)

14. **Q14**: 商數法則 $\frac{d}{dx}(\frac{u}{v}) = $ ？
    - (A) $\frac{u'v - uv'}{v^2}$ (B) $\frac{uv' - u'v}{v^2}$ (C) $\frac{u'}{v'}$ (D) $\frac{u'v + uv'}{v^2}$
    - *答案*: A (W6)

15. **Q15**: 函數 $f(x) = x^3 - 3x$ 的相對極大值發生在 $x = $ ？
    - (A) 0 (B) 1 (C) -1 (D) $\sqrt{3}$
    - *答案*: C (W10)

16. **Q16**: $\int \frac{1}{x} dx = $ ？
    - (A) $\ln|x| + C$ (B) $-1/x^2 + C$ (C) $e^x + C$ (D) 1
    - *答案*: A (W13)

17. **Q17**: 定積分 $\int_{-1}^1 x^3 dx = $ ？
    - (A) 1/2 (B) 0 (C) 1/4 (D) -1/2
    - *答案*: B (W15)

18. **Q18**: 若 $u = 1+x^2$，則 $x dx = $ ？
    - (A) $du$ (B) $2 du$ (C) $\frac{1}{2} du$ (D) $u du$
    - *答案*: C (W16)

19. **Q19**: 使用剝殼法繞 y 軸旋轉 $y=f(x)$，微小體積 $dV = $ ？
    - (A) $\pi f(x)^2 dx$ (B) $2\pi x f(x) dx$ (C) $2\pi f(x) dx$ (D) $\pi x^2 dx$
    - *答案*: B (W17)

20. **Q20**: $f(x) = x^2$ 在 $[0, 2]$ 上的平均值為？
    - (A) 4/3 (B) 2 (C) 4 (D) 8/3
    - *答案*: A (W15)

### 2. 多選題 (Multiple Choice) - 共 20 題

21. **Q21**: 下列哪些極限結果為 $\infty$？
    - (A) $\lim_{x\to 0^+} \frac{1}{x}$ (B) $\lim_{x\to \infty} e^x$ (C) $\lim_{x\to \infty} \frac{1}{x}$ (D) $\lim_{x\to 0} \frac{1}{x^2}$
    - *答案*: A, B, D (W3)

22. **Q22**: 關於連續函數 $f$ 在閉區間 $[a, b]$，正確的是：
    - (A) 必有最大值 (B) 必有最小值 (C) 若 $f(a)f(b) < 0$ 則必有根 (D) 必可微
    - *答案*: A, B, C (W4)

23. **Q23**: 下列微分公式正確的有：
    - (A) $\frac{d}{dx} \tan x = \sec^2 x$ (B) $\frac{d}{dx} e^x = e^x$ (C) $\frac{d}{dx} \ln x = 1/x$ (D) $\frac{d}{dx} \cos x = \sin x$
    - *答案*: A, B, C (W6)

24. **Q24**: 連鎖律的應用實例包括：
    - (A) $\frac{d}{dx} (x^2+1)^5$ (B) $\frac{d}{dx} \sin(e^x)$ (C) $\frac{d}{dx} (x \sin x)$ (D) $\frac{d}{dx} \sqrt{x}$
    - *答案*: A, B (W6)

25. **Q25**: 下列哪些點可能是函數的極值點？
    - (A) $f'(x) = 0$ 的點 (B) $f'(x)$ 不存在的點 (C) 區間端點 (D) $f''(x) = 0$ 的點
    - *答案*: A, B, C (W9)

26. **Q26**: 關於羅必達法則的使用條件：
    - (A) 型式為 $0/0$ (B) 型式為 $\infty/\infty$ (C) 分子分母皆可微 (D) 必須是多項式
    - *答案*: A, B, C (W11)

27. **Q27**: 反導數的性質包括：
    - (A) 兩個反導數之間相差一個常數 (B) 不定積分是所有反導數的集合 (C) 積分是微分的逆運算 (D) 所有函數都有初等函數形式的反導數
    - *答案*: A, B, C (W13)

28. **Q28**: 微積分基本定理 (FTC) 連結了：
    - (A) 微分 (B) 積分 (C) 極限 (D) 面積
    - *答案*: A, B, D (W15)

29. **Q29**: 使用代換法時需注意：
    - (A) 找到合適的 $u$ (B) 換算 $dx$ 為 $du$ (C) 定積分需換上下限 (D) 最後必須換回 $x$ (定積分除外)
    - *答案*: A, B, C, D (W16)

30. **Q30**: 求旋轉體體積的方法有：
    - (A) 圓盤法 (B) 墊圈法 (C) 剝殼法 (D) 黎曼和法
    - *答案*: A, B, C (W17)

31. **Q31**: 下列哪些極限可以使用羅必達法則直接或變形後處理？
    - (A) $\lim x \ln x$ (B) $\lim \frac{\sin x}{x}$ (C) $\lim 1^\infty$ (D) $\lim (x - \ln x)$
    - *答案*: A, B, C (W11)

32. **Q32**: 函數 $f(x) = x^2$ 在其定義域內是：
    - (A) 偶函數 (B) 始終遞增 (C) 向上凹 (D) 有最小值
    - *答案*: A, C, D (W10)

33. **Q33**: 定積分的性質包括：
    - (A) 線性性質 (B) 區間可加性 (C) 比較性質 (D) $\int_a^a f dx = 0$
    - *答案*: A, B, C, D (W14)

34. **Q34**: 下列哪些是偶函數？
    - (A) $\cos x$ (B) $x^2$ (C) $|x|$ (D) $\sin x$
    - *答案*: A, B, C (W16)

35. **Q35**: 剝殼法相對於圓盤法的優點：
    - (A) 繞 y 軸旋轉時不一定要解 $x$ 為 $y$ 的函數 (B) 幾何結構有時更簡單 (C) 適用範圍更廣 (D) 精度更高
    - *答案*: A, B (精度取決於解析計算，兩者一致) (W17)

36. **Q36**: 影響函數連續性的因素有：
    - (A) 分母為零 (B) 根號內為負 (C) 分段點極限不相等 (D) 函數值未定義
    - *答案*: A, C, D (W2)

37. **Q37**: 導數的幾何與物理意義：
    - (A) 切線斜率 (B) 瞬時速度 (C) 邊際成本 (D) 總變量
    - *答案*: A, B, C (W5)

38. **Q38**: 反曲點 (Inflection Point) 的特徵：
    - (A) 凹凸性改變 (B) $f''(x) = 0$ 或不存在 (C) 切線垂直 (D) 函數值最大
    - *答案*: A, B (W10)

39. **Q39**: 不定積分 $\int \sin x dx$ 的結果可以是：
    - (A) $-\cos x + 5$ (B) $-\cos x - 10$ (C) $\cos x + C$ (D) $-\cos x + C$
    - *答案*: A, B, D (W13)

40. **Q40**: 曲線間面積計算時需確定：
    - (A) 上下函數關係 (B) 積分上下限（交點） (C) 旋轉軸 (D) 是否對稱
    - *答案*: A, B, D (面積不需旋轉軸) (W17)

### 3. 填充題 (Fill-in-the-blank) - 共 10 題

41. **Q41**: $\lim_{x\to 0} \frac{\sin 5x}{x} = $ __________。
    - *答案*: 5 (W1)

42. **Q42**: $f(x) = \ln(x^2+1)$，則 $f'(0) = $ __________。
    - *答案*: 0 (W6)

43. **Q43**: 若 $xy = 1$，則 $dy/dx = $ __________。
    - *答案*: -y/x (或 -1/x^2) (W7)

44. **Q44**: 函數 $f(x) = x^4$ 的二階導數 $f''(x) = $ __________。
    - *答案*: 12x^2 (W10)

45. **Q45**: 使用羅必達法則，$\lim_{x\to \infty} \frac{\ln x}{x} = $ __________。
    - *答案*: 0 (W11)

46. **Q46**: $\int_1^2 \frac{1}{x^2} dx = $ __________。
    - *答案*: 1/2 (W15)

47. **Q47**: 代換積分 $\int 2x e^{x^2} dx = $ __________ $+ C$。
    - *答案*: e^{x^2} (W16)

48. **Q48**: 半徑為 $R$ 的圓面積公式 $A = \pi R^2$ 可由積分 $\int_{-R}^R 2\sqrt{R^2-x^2} dx$ 導出，這體現了 __________ 的應用。
    - *答案*: 面積 (或 定積分) (W17)

49. **Q49**: 微積分基本定理第一部分：$\frac{d}{dx} \int_a^x f(t) dt = $ __________。
    - *答案*: f(x) (W15)

50. **Q50**: 旋轉體體積：若區域在 $x$ 軸上方，繞 $x$ 軸旋轉一周，橫截面是 __________。
    - *答案*: 圓形 (W17)
