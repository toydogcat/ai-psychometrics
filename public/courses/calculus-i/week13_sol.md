## 二、 詳細解析 (Detailed Analysis)

### 1. 單選題
*   **Q1**: 若在開區間內只有一個臨界點且 $f''(x) > 0$（凹向上），則該點不僅是局部極小，也是該區間內的絕對最小值。
*   **Q2**: 體積 $V = x(L-2x)^2$。求導 $V' = (L-2x)^2 + x \cdot 2(L-2x)(-2) = (L-2x)(L-2x-4x) = (L-2x)(L-6x)$。令 $V'=0 \implies x = L/6$ ($x=L/2$ 會使體積為 0)。
*   **Q6**: 牛頓法公式 $x_{n+1} = x_n - \frac{x_n^2 - a}{2x_n} = \frac{2x_n^2 - x_n^2 + a}{2x_n} = \frac{x_n^2 + a}{2x_n} = \frac{1}{2}(x_n + \frac{a}{x_n})$。當 $a=2$ 時，即為選項 (A)。
*   **Q9**: $y = \int e^x dx = e^x + C$。代入 $(0, 2) \implies e^0 + C = 2 \implies 1 + C = 2 \implies C = 1$。故 $y = e^x + 1$。

### 2. 多選題
*   **Q12**: 對於固定體積圓柱體，最小表面積發生在 $h=2r$。此時側面積 $2\pi rh = 4\pi r^2$，而單個底面積為 $\pi r^2$。因此側面積是單個底面積的 4 倍，是總底面積（上下兩底）的 2 倍。
*   **Q18**: 對於直線 $f(x) = ax+b$，切線即為直線本身。$x_1 = x_0 - \frac{ax_0 + b}{a} = x_0 - x_0 - \frac{b}{a} = -b/a$。這正是方程的根，故只需一次迭代。
*   **Q19**: (D) 正確是因為當 $R(x) > C(x)$ 時代表有利潤，而若此時 $P'(x) > 0$（即 $MR > MC$），增加產量會使利潤進一步增加。

### 3. 填充題
*   **Q21**: $\int (2x+3) dx = 2(\frac{x^2}{2}) + 3x + C = x^2 + 3x + C$。
*   **Q25**: 該公式 $x_{n+1} = \frac{1}{2}(x_n + \frac{a}{x_n})$ 被稱為「巴比倫法」或「希羅法」，用於手工計算平方根。
*   **Q27**: 速度是加速度的反導數：$v(t) = \int -9.8 dt = -9.8t + C$。代入初始條件 $v(0) = v_0 \implies C = v_0$。

## 四、 總結 (Summary)
第 13 週將導數的應用推向頂峰，並開啟了積分學的大門。
1. **建模是關鍵**：最佳化問題最難的部分不在於求導，而在於如何將文字題轉化為數學式。務必注意變數的物理意義與範圍。
2. **數值方法的優勢**：許多實際方程（如超越方程）無法求得精確解，牛頓法提供了一種極其高效的逼近途徑。
3. **反向思考**：反導數要求我們反向運用微分規則。理解 $F(x)+C$ 中的 $C$ 代表了整個函數族群，是學習微積分基本定理的重要基礎。
