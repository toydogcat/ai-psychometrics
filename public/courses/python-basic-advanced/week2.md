## 一、 單元講解 (Lecture) - 總計 100 分鐘

### 1. 變數賦值與記憶體模型 (20 min)
*   **賦值的本質 (KP2.1)**：在 Python 中，變數更像是「標籤」而非「容器」。賦值是將標籤貼在記憶體中的物件上。
*   **關鍵函數**：
    *   `id()`：獲取物件在記憶體中的唯一標識（地址）。
    *   `type()`：查看變數目前指向的物件型態。
*   **動態型別**：同一個變數名可以先指向整數，再指向字串。

### 2. 數值型態與算術運算 (20 min)
*   **常用數值 (KP2.2)**：
    *   `int` (整數)：無長度限制。
    *   `float` (浮點數)：科學計數法與精度問題。
*   **運算符號**：
    *   基礎：`+`, `-`, `*`, `/` (除法永遠返回浮點數)。
    *   進階：`//` (整除), `%` (取餘數), `**` (次方)。
*   **複合賦值**：`x += 1`, `y *= 2`。

### 3. 字串操作與切片 (25 min)
*   **索引與切片 (KP2.3)**：
    *   正向索引從 0 開始，反向索引從 -1 開始。
    *   語法：`s[start:stop:step]` (包含 start，不包含 stop)。
*   **常用方法**：
    *   `.strip()`：去除兩端空格。
    *   `.lower()`, `.upper()`：大小寫轉換。
    *   `.split()`, `.join()`：字串切割與連結。
    *   `.replace()`：替換內容。

### 4. 布林邏輯與比較運算 (20 min)
*   **布林值 (KP2.4)**：`True` 與 `False` (首字母必須大寫)。
*   **比較運算**：`==`, `!=`, `<`, `>`, `<=`, `>=`。
*   **邏輯運算**：`and` (且), `or` (或), `not` (非)。
*   **Truthy & Falsy**：在 Python 中，`0`, `""`, `[]`, `None` 在布林環境下都被視為 `False`。

### 5. 型別轉換與內建函數 (15 min)
*   **顯式轉換 (KP2.5)**：`int()`, `float()`, `str()`, `bool()`。
*   **常用函數 (KP2.6)**：
    *   `len()`：計算長度（字串、容器）。
    *   `round()`：四捨五入。
    *   `abs()`：絕對值。

## 二、 動手實作 (Lab) - 總計 50 分鐘

### 實作一：變數追蹤與記憶體觀察 (15 min)
**任務目標**：理解變數指向物件的概念。
1.  在 Colab 中輸入以下代碼並觀察輸出：
    ```python
    a = [1, 2, 3]
    b = a
    print(f"a id: {id(a)}, b id: {id(b)}")
    print(f"a is b: {a is b}")
    
    a.append(4)
    print(f"b contents: {b}") # 觀察 b 是否跟著改變
    ```
2.  思考：為什麼修改 `a` 會影響到 `b`？

### 實作二：字串魔術師 (20 min)
**任務目標**：練習字串切片與方法。
1.  給定一個髒字串：`raw_data = "  Python_Programming_Is_Fun  "`。
2.  任務：
    - 去除首尾空格。
    - 將所有字母轉為小寫。
    - 將底線 `_` 替換為空格。
    - 取得「programming」這個單詞。
    ```python
    # 參考解答
    cleaned = raw_data.strip().lower().replace("_", " ")
    words = cleaned.split()
    print(words[1])
    ```

### 實作三：單位轉換器 (15 min)
**任務目標**：綜合運用數值運算與型別轉換。
1.  編寫程式，讓用戶輸入攝氏溫度 (Celsius)。
2.  計算並輸出華氏溫度 (Fahrenheit)，公式為：`F = C * 9/5 + 32`。
3.  輸出結果需四捨五入到小數點後一位。
