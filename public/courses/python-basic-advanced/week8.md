# 課程一：第 8 週 - 函式設計範式 (Functional Programming)

本文件包含了第 8 週完整的教學大綱、實作指南以及練習題庫。

---

## 一、 單元講解 (Lecture) - 總計 100 分鐘

### 1. 匿名函數 (Lambda Functions) (20 min)
*   **概念與語法 (KP8.1)**：`lambda arguments: expression`。
*   **特性**：
    *   無名、單行、自動回傳表達式的結果。
    *   常用於需要臨時小函數的場景，如排序 `key`。
*   **限制**：不能包含多行邏輯或複雜語句（如 `if-elif-else` 的完整形式，僅限三元運算子）。

### 2. 高階函數：Map & Filter (20 min)
*   **Map (KP8.2)**：將一個函數應用到序列的所有元素。
    *   `map(func, iterable)`
*   **Filter (KP8.3)**：過濾出序列中符合條件（函數回傳為 True）的元素。
    *   `filter(func, iterable)`
*   **特性**：在 Python 3 中，這兩者都回傳「疊代器」(Iterator)，節省記憶體。

### 3. Reduce 與偏函數 (Partial Functions) (20 min)
*   **Reduce (KP8.4)**：對序列進行累計運算。
    *   需要從 `functools` 匯入。
    *   `reduce(func, iterable[, initializer])`
*   **Partial**：固定函數的部分參數，產生一個新函數。

### 4. 產生器表達式 (Generator Expressions) (20 min)
*   **語法 (KP8.5)**：`(expression for item in iterable if condition)`。
*   **List vs Generator**：
    *   List Comprehension: 立即生成完整串列，佔用記憶體。
    *   Generator: 延遲計算 (Lazy Evaluation)，一次產生一個值，適合處理大數據。
*   **`next()` 函數**：手動獲取產生器的下一個值。

### 5. 閉包與裝飾器基礎 (20 min)
*   **閉包 (Closure)**：內層函數引用了外層函數的變數。
*   **裝飾器 (Decorator)**：修改或增強函數功能的強大工具。
    *   基礎語法：`@decorator_name`。
    *   原理：函數作為一等公民 (First-class citizen) 的應用。

## 二、 動手實作 (Lab) - 總計 50 分鐘

### 實作一：資料清理管道 (20 min)
**任務目標**：綜合運用 map, filter 與 lambda。
1.  給定一份原始資料 `raw_data = [" 123 ", "abc", " 456", "789 ", "def"]`。
2.  使用 `filter` 移除所有包含字母的字串（僅保留數字字串）。
3.  使用 `map` 將剩餘字串去除空格並轉為整數。
4.  將結果轉為串列並印出。

### 實作二：自定義排序邏輯 (15 min)
**任務目標**：練習 Lambda 在排序中的應用。
1.  建立一個包含字典的串列，代表學生成績：`students = [{"name": "A", "score": 80}, {"name": "B", "score": 95}, {"name": "C", "score": 80}]`。
2.  任務：
    - 先按分數由高到低排序。
    - 若分數相同，則按名字字母順序排序。
    - 使用 `lambda` 作為 `key` 參數。

### 實作三：無限序列產生器概念 (15 min)
**任務目標**：理解產生器的延遲計算。
1.  建立一個產生器表達式，計算 1 到 100 萬的平方值。
2.  觀察記憶體佔用（與 List Comprehension 比較）。
3.  使用 `for` 迴圈印出前 10 個結果，並說明為什麼它不會讓電腦當機。

## 四、 課後測驗題庫 (Quiz) - 30 分鐘

### 1. 單選題 (Single Choice) - 共 10 題

1. **Q1**: Lambda 函數的核心特點是？
   - (A) 可以有多行程式碼 (B) 匿名且只能包含一個表達式 (C) 執行速度比 def 快很多 (D) 不需要參數
   - *答案*: B (KP8.1)

2. **Q2**: `map(lambda x: x*2, [1, 2, 3])` 在 Python 3 的直接回傳值型態是？
   - (A) [2, 4, 6] (B) (2, 4, 6) (C) map object (iterator) (D) None
   - *答案*: C (KP8.2)

3. **Q3**: 想要從串列中過濾出偶數，應使用哪個函數？
   - (A) map (B) reduce (C) filter (D) apply
   - *答案*: C (KP8.3)

4. **Q4**: `reduce` 函數位於哪個標準庫模組中？
   - (A) itertools (B) functools (C) math (D) collections
   - *答案*: B (KP8.4)

5. **Q5**: 產生器表達式使用哪種括號？
   - (A) [] (B) {} (C) () (D) <>
   - *答案*: C (KP8.5)

6. **Q6**: 關於延遲計算 (Lazy Evaluation)，下列描述正確的是？
   - (A) 程式執行變慢了 (B) 僅在需要時才計算下一個值 (C) 會佔用更多記憶體 (D) 只適用於小數據
   - *答案*: B (KP8.5)

7. **Q7**: `(lambda x, y: x if x > y else y)(5, 10)` 的結果是？
   - (A) 5 (B) 10 (C) True (D) False
   - *答案*: B (KP8.1)

8. **Q8**: 如何手動取得疊代器的下一個元素？
   - (A) get() (B) next() (C) continue() (D) move()
   - *答案*: B (KP8.5)

9. **Q9**: `list(filter(None, [0, 1, False, True]))` 的結果是？
   - (A) [0, 1, False, True] (B) [1, True] (C) [0, False] (D) []
   - *答案*: B (KP8.3)

10. **Q10**: 關於 `reduce(lambda x, y: x + y, [1, 2, 3, 4])` 的執行過程，第一個運算是？
    - (A) 1 + 2 (B) 1 + 2 + 3 + 4 (C) 1 + 4 (D) 0 + 1
    - *答案*: A (KP8.4)

### 2. 多選題 (Multiple Choice) - 共 10 題

1. **Q11**: 哪些是 Lambda 函數的合法寫法？
   - (A) `lambda: "Hello"` (B) `lambda x, y: x + y` (C) `lambda x: print(x)` (D) `lambda x: x > 0`
   - *答案*: A, B, C, D (KP8.1)

2. **Q12**: 關於 `map` 與 `filter` 的共同點，哪些正確？
   - (A) 都回傳疊代器 (B) 都不會修改原始資料 (C) 都接受一個函數作為參數 (D) 執行結果長度一定相同
   - *答案*: A, B, C (KP8.2, KP8.3)

3. **Q13**: 產生器 (Generator) 的優點包括？
   - (A) 節省記憶體 (B) 支持無限序列 (C) 可以反向疊代 (D) 代碼通常比傳統迴圈簡潔
   - *答案*: A, B, D (KP8.5)

4. **Q14**: 下列哪些函數屬於「高階函數」(Higher-order functions)？
   - (A) map (B) filter (C) sorted (D) reduce
   - *答案*: A, B, C, D (KP8.2, KP8.3, KP8.4)

5. **Q15**: 關於 `reduce` 的 `initializer` 參數，哪些正確？
   - (A) 是選用參數 (B) 若提供，則作為運算的起始值 (C) 若串列為空且有提供，則直接回傳它 (D) 必須是整數
   - *答案*: A, B, C (KP8.4)

6. **Q16**: 下列哪些操作會將產生器「耗盡」(Exhausted)？
   - (A) 將其轉為 list (B) 使用 for 迴圈遍歷完畢 (C) 使用 next() 直到報錯 (D) 印出產生器物件本身
   - *答案*: A, B, C (KP8.5)

7. **Q17**: 哪些場景適合使用 Lambda？
   - (A) 簡單的數學轉換 (B) 串列排序的 key 參數 (C) GUI 介面的事件回呼 (Callback) (D) 複雜的業務邏輯處理
   - *答案*: A, B, C (KP8.1)

8. **Q18**: 關於 `list(map(str, [1, 2, 3]))`，哪些正確？
   - (A) 結果是 ["1", "2", "3"] (B) 使用了內建函數 str (C) 這是將數字串列轉為字串串列的常用方法 (D) map 會立即執行所有轉換
   - *答案*: A, B, C (KP8.2)

9. **Q19**: 產生器與串列 (List) 的差異有哪些？
   - (A) 產生器不支持索引訪問 (如 `g[0]`) (B) 產生器只能遍歷一次 (C) 產生器使用 `len()` 會報錯 (D) 產生器可以使用 `append()`
   - *答案*: A, B, C (KP8.5)

10. **Q20**: 閉包 (Closure) 的特徵包括？
    - (A) 必須有巢狀函數 (B) 內層函數引用了外層變數 (C) 外層函數回傳了內層函數 (D) 變數會被保存在記憶體中
    - *答案*: A, B, C, D (Lecture contents)

### 3. 填充題 (Fill-in-the-blank) - 共 10 題

1. **Q21**: Lambda 函數又被稱為 __________ 函數。
   - *答案*: 匿名 (KP8.1)

2. **Q22**: `map` 函數的作用是將一個函數 __________ 到序列的每個元素上。
   - *答案*: 映射 (或 應用) (KP8.2)

3. **Q23**: 想要將 `[1, 2, 3]` 累加成 `6`，除了用 `sum`，也可以用 `__________` 函數。
   - *答案*: reduce (KP8.4)

4. **Q24**: 產生器物件在 Python 中屬於 __________ (Iterator) 的一種。
   - *答案*: 疊代器 (KP8.5)

5. **Q25**: `filter(lambda x: x > 5, [3, 6, 9])` 過濾後的結果（轉為 list）是 __________。
   - *答案*: [6, 9] (KP8.3)

6. **Q26**: 產生器表達式比串列生成式更省 __________ (空間資源)。
   - *答案*: 記憶體 (KP8.5)

7. **Q28**: 想要固定一個函數的某些參數，可以使用 `functools.__________`。
   - *答案*: partial (Lecture contents)

8. **Q29**: `(lambda x: x**2)(4)` 的結果是 __________。
   - *答案*: 16 (KP8.1)

9. **Q30**: 當疊代器沒有下一個值時，呼叫 `next()` 會引發 __________ 異常。
   - *答案*: StopIteration (KP8.5)

10. **Q30**: 函數內部定義了另一個函數，且引用了外部變數，這種結構稱為 __________ (Closure)。
    - *答案*: 閉包 (Lecture contents)
