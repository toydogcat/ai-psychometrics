# 課程一：第 7 週 - 函數與模組 (Functions & Modules)

本文件包含了第 7 週完整的教學大綱、實作指南以及練習題庫。

---

## 一、 單元講解 (Lecture) - 總計 100 分鐘

### 1. 函數定義與參數傳遞 (20 min)
*   **基礎語法 (KP7.1)**：使用 `def` 關鍵字，縮進編寫代碼塊。
*   **參數類型**：
    *   **位置參數 (Positional)**：按順序傳入。
    *   **關鍵字參數 (Keyword)**：指定 `name=value`。
    *   **預設參數**：`def func(a=10):` (注意：不可變物件作為預設值的陷阱)。
*   **回傳值 (Return)**：可以回傳單一值、多個值 (Tuple) 或 `None`。

### 2. 不定長度參數 (*args, **kwargs) (20 min)
*   **`*args` (KP7.2)**：收集多餘的位置參數，封裝成 Tuple。
*   **`**kwargs`**：收集多餘的關鍵字參數，封裝成 Dictionary。
*   **參數解包 (Unpacking)**：呼叫函數時使用 `*` 或 `**` 將容器展開。

### 3. 變數作用域 (Scope) (20 min)
*   **LEGB 原則 (KP7.3)**：
    *   **L**ocal (區域), **E**nclosing (閉包外層), **G**lobal (全域), **B**uilt-in (內建)。
*   **`global` 與 `nonlocal`**：何時需要宣告這些關鍵字來修改外層變數。

### 4. 模組匯入與命名空間 (20 min)
*   **匯入方式 (KP7.4)**：
    *   `import module`
    *   `from module import function`
    *   `import module as alias`
*   **`__name__ == "__main__"`**：區分模組是被匯入還是直接執行。
*   **自定義模組**：建立 `.py` 檔案並在其他地方匯入。

### 5. 常用標準函數庫 (Standard Library) (20 min)
*   **`os`, `sys`**：系統操作與環境變數。
*   **`math`, `random`**：數學運算與隨機數生成。
*   **`datetime`**：日期與時間處理。
*   **`json`**：資料格式交換。

## 二、 動手實作 (Lab) - 總計 50 分鐘

### 實作一：多功能計算器模組 (20 min)
**任務目標**：練習函數定義與模組匯入。
1.  建立一個檔案 `my_math.py`，包含加、減、乘、除四個函數。
2.  在主程式 `main.py` 中匯入 `my_math` 並呼叫其功能。
3.  確保 `my_math.py` 中有測試代碼，且只有在直接執行時才會跑測試。

### 實作二：靈活的打招呼函數 (15 min)
**任務目標**：練習 `*args` 與 `**kwargs`。
1.  寫一個函數 `greet(*names, **details)`。
2.  `names` 可以接收多個名字，並對每個人說 Hello。
3.  `details` 接收如 `title="Manager"`, `location="Taipei"` 等資訊並印出。
4.  測試：`greet("Alice", "Bob", title="Engineers")`。

### 實作三：猜數字遊戲加強版 (15 min)
**任務目標**：練習 `random` 模組與作用域。
1.  使用 `random.randint(1, 100)` 生成目標數字。
2.  使用一個全域變數記錄「最高分」(最少次數)。
3.  建立函數進行遊戲，若打破記錄，使用 `global` 更新最高分。

## 四、 課後測驗題庫 (Quiz) - 30 分鐘

### 1. 單選題 (Single Choice) - 共 10 題

1. **Q1**: 下列哪一個關鍵字用於定義 Python 函數？
   - (A) function (B) def (C) func (D) lambda
   - *答案*: B (KP7.1)

2. **Q2**: 若函數內沒有 `return` 語句，其返回值為？
   - (A) 0 (B) False (C) None (D) 報錯
   - *答案*: C (KP7.1)

3. **Q3**: `def f(a, b=2, c=3):`，呼叫 `f(1, c=5)` 時，參數 `b` 的值是？
   - (A) 1 (B) 2 (C) 5 (D) 報錯
   - *答案*: B (KP7.1)

4. **Q4**: 關於 `*args`，它在函數內部會被轉換成什麼型態？
   - (A) List (B) Dictionary (C) Tuple (D) Set
   - *答案*: C (KP7.2)

5. **Q5**: 在函數內部修改全域變數 `x`，必須先宣告？
   - (A) `extern x` (B) `global x` (C) `public x` (D) `local x`
   - *答案*: B (KP7.3)

6. **Q6**: LEGB 原則中的 **E** 代表什麼？
   - (A) Exterior (B) Enclosing (C) Empty (D) Element
   - *答案*: B (KP7.3)

7. **Q7**: 如何從 `math` 模組中僅匯入 `sqrt` 函數？
   - (A) `import math.sqrt` (B) `from math import sqrt` (C) `import sqrt from math` (D) `math.import(sqrt)`
   - *答案*: B (KP7.4)

8. **Q8**: 哪一個標準庫可以用來生成隨機整數？
   - (A) random (B) math (C) os (D) sys
   - *答案*: A (KP7.5)

9. **Q9**: `__name__` 變數在模組被直接執行時，其值為？
   - (A) "module" (B) "__main__" (C) "filename" (D) "python"
   - *答案*: B (KP7.4)

10. **Q10**: 執行 `print(type(json.loads('{"a": 1}')))` 的結果是？
    - (A) <class 'str'> (B) <class 'dict'> (C) <class 'json'> (D) <class 'list'>
    - *答案*: B (KP7.5)

### 2. 多選題 (Multiple Choice) - 共 10 題

1. **Q11**: 關於函數參數，哪些描述正確？
   - (A) 預設參數必須放在非預設參數之後 (B) 呼叫時位置參數必須在關鍵字參數之前 (C) 預設參數可以是任何型態 (D) 同一個函數可以回傳不同型態的值
   - *答案*: A, B, C, D (KP7.1)

2. **Q12**: 下列哪些是 `**kwargs` 的正確用法？
   - (A) 接收任意數量的關鍵字參數 (B) 內部型態為字典 (C) 必須定義在 `*args` 之後 (D) 參數名必須叫 kwargs
   - *答案*: A, B, C (KP7.2)

3. **Q13**: 關於 LEGB 作用域，哪些正確？
   - (A) 搜尋順序由內向外 (B) Built-in 是最後一層搜尋 (C) Local 變數在函數結束後消失 (D) 在 Local 可以直接讀取 Global 變數
   - *答案*: A, B, C, D (KP7.3)

4. **Q14**: 模組匯入的優點包括？
   - (A) 代碼重用 (B) 邏輯分離 (C) 避免命名衝突 (使用 alias) (D) 讓程式跑得更快
   - *答案*: A, B, C (KP7.4)

5. **Q15**: 哪些方法可以將 `[1, 2, 3]` 作為參數傳給 `func(a, b, c)`？
   - (A) `func([1, 2, 3])` (B) `func(1, 2, 3)` (C) `L=[1,2,3]; func(*L)` (D) `func(a=1, b=2, c=3)`
   - *答案*: B, C, D (KP7.1, KP7.2)

6. **Q16**: `os` 模組可以執行哪些操作？
   - (A) 取得當前工作目錄 (B) 列出資料夾檔案 (C) 修改檔案路徑 (D) 計算三角函數
   - *答案*: A, B, C (KP7.5)

7. **Q17**: 關於預設參數使用可變物件 (如 `L=[]`) 的後果？
   - (A) 每次呼叫都會建立新串列 (B) 串列物件在函數定義時僅建立一次 (C) 多次呼叫會共用同一個串列 (D) 這是推薦的程式寫法
   - *答案*: B, C (KP7.1)

8. **Q18**: 下列哪些屬於 Python 的 Built-in (內建) 作用域？
   - (A) print (B) len (C) int (D) 自己定義的變數
   - *答案*: A, B, C (KP7.3)

9. **Q19**: 如何取得目前的日期與時間？
   - (A) `datetime.now()` (B) `datetime.datetime.now()` (需要正確匯入) (C) `time.time()` (D) `os.get_time()`
   - *答案*: B, C (KP7.5)

10. **Q20**: `if __name__ == "__main__":` 的作用？
    - (A) 防止模組內容被重複執行 (B) 封裝模組的測試代碼 (C) 讓檔案既可作為腳本執行也可作為模組匯入 (D) 提高代碼安全性
    - *答案*: B, C (KP7.4)

### 3. 填充題 (Fill-in-the-blank) - 共 10 題

1. **Q21**: 函數定義中，接收任意數量位置參數的語法是 __________。
   - *答案*: *args (KP7.2)

2. **Q22**: 想要在區域作用域修改全域變數，需要使用 __________ 關鍵字。
   - *答案*: global (KP7.3)

3. **Q23**: Python 搜尋變數的順序規則簡稱為 __________。
   - *答案*: LEGB (KP7.3)

4. **Q24**: 將字典 `d = {"a": 1, "b": 2}` 解包傳遞給函數的語法是 `func(__________)`。
   - *答案*: **d (KP7.2)

5. **Q25**: 模組 `random` 中，產生 0 到 1 之間浮點數的函數是 __________。
   - *答案*: random() (KP7.5)

6. **Q26**: 想要幫匯入的模組取別名，使用關鍵字 __________。
   - *答案*: as (KP7.4)

7. **Q27**: `sys.path` 是一個串列，包含了 Python 搜尋 __________ 的路徑。
   - *答案*: 模組 (KP7.5)

8. **Q28**: 函數執行完畢後，可以使用 __________ 關鍵字將結果傳回給呼叫者。
   - *答案*: return (KP7.1)

9. **Q29**: JSON 模組中，將 Python 物件轉為 JSON 字串的方法是 __________。
   - *答案*: dumps() (KP7.5)

10. **Q30**: 一個包含 Python 代碼並以 `.py` 結尾的檔案稱為一個 __________。
    - *答案*: 模組 (KP7.4)
