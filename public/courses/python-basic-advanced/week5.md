## 一、 單元講解 (Lecture) - 總計 100 分鐘

### 1. 串列基礎與可變性 (Mutability) (20 min)
*   **串列 (List) 特性 (KP5.1)**：
    *   有序的集合，可以包含不同型別的物件。
    *   **可變性 (Mutable)**：可以直接修改元素內容、增加或刪除元素。
*   **記憶體配置**：串列儲存的是物件的引用 (Reference)。
*   **常用操作**：`.append()`, `.extend()`, `.insert()`, `.pop()`, `.remove()`。

### 2. 串列生成式 (List Comprehension) (20 min)
*   **語法結構 (KP5.2)**：`[expression for item in iterable if condition]`。
*   **優勢**：比傳統 `for` 迴圈更簡潔且執行效率通常更高。
*   **應用場景**：資料過濾、數值轉換、建立特定規律的串列。

### 3. 元組特性與解構賦值 (Tuple Unpacking) (20 min)
*   **元組 (Tuple) 特性 (KP5.3)**：
    *   **不可變性 (Immutable)**：一旦建立就不能修改內容。
    *   佔用記憶體較小，執行速度較快，常作為不可變的資料記錄。
*   **解構賦值 (Unpacking)**：
    *   `a, b = (10, 20)`。
    *   使用 `*` 收集多餘元素：`first, *middle, last = [1, 2, 3, 4, 5]`。

### 4. 巢狀結構與深淺拷貝 (20 min)
*   **巢狀串列 (Nesting) (KP5.4)**：串列中包含串列，用於表示矩陣或多維資料。
*   **淺拷貝 (Shallow Copy)**：`list.copy()` 或 `list[:]`，僅複製第一層引用。
*   **深拷貝 (Deep Copy)**：使用 `copy.deepcopy()`，遞迴複製所有層級的物件。

### 5. 排序與搜尋 (20 min)
*   **排序 (KP5.5)**：
    *   `.sort()`：就地排序 (In-place)，修改原串列。
    *   `sorted()`：回傳新串列，不影響原物件。
    *   **Key 參數**：使用 `key=len` 或 `key=str.lower` 進行自定義排序。
*   **搜尋與統計**：`in` 運算子、`.index()`, `.count()`。

## 二、 動手實作 (Lab) - 總計 50 分鐘

### 實作一：學生成績管理系統 (20 min)
**任務目標**：練習串列的基本操作與排序。
1.  建立一個串列 `grades = [85, 92, 78, 90, 88]`。
2.  新增一個學生成績 95 分。
3.  刪除最低分的成績。
4.  將成績由高到低排序並輸出。
5.  計算平均分數。

### 實作二：九九乘法表生成器 (15 min)
**任務目標**：練習巢狀串列與串列生成式。
1.  使用串列生成式建立一個 9x9 的巢狀串列，其中每個元素為 `i * j` 的結果。
2.  印出第三列 (3x1 到 3x9) 的結果。

### 實作三：座標互換與解構 (15 min)
**任務目標**：練習 Tuple 解構與應用。
1.  給定一個點的座標 `point = (10, 20, 30)`。
2.  使用解構賦值將 `x, y, z` 分別提取出來。
3.  寫一個交換函數的概念，利用 Tuple 特性一行程式碼交換 `a` 與 `b` 的值。

## 四、 課後測驗題庫 (Quiz) - 30 分鐘

### 1. 單選題 (Single Choice) - 共 10 題

1. **Q1**: 下列哪種資料型別是「不可變」(Immutable) 的？
   - (A) List (B) Tuple (C) Dictionary (D) Set
   - *答案*: B (KP5.3)

2. **Q2**: 執行 `my_list = [1, 2]; my_list.append([3, 4])` 後，串列的長度是？
   - (A) 3 (B) 4 (C) 2 (D) 報錯
   - *答案*: A (KP5.1)

3. **Q3**: 下列哪個表達式可以產生 `[0, 4, 16]`？
   - (A) `[x*2 for x in range(3)]` (B) `[x**2 for x in range(0, 5, 2)]` (C) `[x**2 for x in range(3)]` (D) `[x^2 for x in [0, 2, 4]]`
   - *答案*: B (KP5.2)

4. **Q4**: `a, b = (1, 2, 3)` 會導致什麼結果？
   - (A) a=1, b=2 (B) a=1, b=[2, 3] (C) ValueError (D) a=1, b=3
   - *答案*: C (KP5.3)

5. **Q5**: 如何在串列 `L` 的索引 1 位置插入元素 "X"？
   - (A) L.add(1, "X") (B) L.insert(1, "X") (C) L.append(1, "X") (D) L[1] = "X"
   - *答案*: B (KP5.1)

6. **Q6**: `list(range(5))[:2]` 的結果是？
   - (A) [0, 1, 2] (B) [1, 2] (C) [0, 1] (D) [0, 2]
   - *答案*: C (KP5.1)

7. **Q7**: 關於 `sorted_list = my_list.sort()`，下列描述正確的是？
   - (A) `sorted_list` 是排序後的串列 (B) `my_list` 維持原樣 (C) `sorted_list` 的值是 `None` (D) 程式會報錯
   - *答案*: C (KP5.5)

8. **Q8**: 若 `nested = [[1], [2, 3]]`, 則 `nested[1][0]` 是？
   - (A) 1 (B) 2 (C) 3 (D) [2, 3]
   - *答案*: B (KP5.4)

9. **Q9**: 使用哪種方法可以將兩個串列合併（不產生巢狀）？
   - (A) .append() (B) .extend() (C) .add() (D) .push()
   - *答案*: B (KP5.1)

10. **Q10**: `(1,)` 的型態是？
    - (A) 整數 (B) 元組 (C) 串列 (D) 語法錯誤
    - *答案*: B (KP5.3)

### 2. 多選題 (Multiple Choice) - 共 10 題

1. **Q11**: 關於串列 (List) 與元組 (Tuple)，哪些描述正確？
   - (A) 串列使用 `[]`，元組使用 `()` (B) 串列是不可變的 (C) 元組可以用作字典的 Key (D) 兩者都支持切片操作
   - *答案*: A, C, D (KP5.1, KP5.3)

2. **Q12**: 下列哪些操作會修改原始串列？
   - (A) .reverse() (B) .sort() (C) .pop() (D) sorted()
   - *答案*: A, B, C (KP5.1, KP5.5)

3. **Q13**: 關於 `new_list = old_list[:]`，哪些正確？
   - (A) 這是淺拷貝 (B) 這是深拷貝 (C) 修改 `new_list` 的基本元素不會影響 `old_list` (D) 這是建立了一個新物件
   - *答案*: A, C, D (KP5.4)

4. **Q14**: 下列哪些是有效的串列生成式語法？
   - (A) `[x for x in range(10)]` (B) `[x if x > 5 else 0 for x in range(10)]` (C) `[x for x in range(10) if x % 2 == 0]` (D) `for x in range(10) yield x`
   - *答案*: A, B, C (KP5.2)

5. **Q15**: 使用 `*` 進行解構賦值，哪些正確？
   - (A) `a, *b = [1, 2, 3]` 則 `b` 為 `[2, 3]` (B) `*a, b = [1, 2, 3]` 則 `a` 為 `[1, 2]` (C) `a, *b, c = [1]` 會報錯 (D) 一個賦值語句中只能有一個 `*`
   - *答案*: A, B, D (KP5.3)

6. **Q16**: 如何刪除串列中的元素？
   - (A) `del L[0]` (B) `L.remove(value)` (C) `L.pop()` (D) `L.clear()`
   - *答案*: A, B, C, D (KP5.1)

7. **Q17**: 關於巢狀串列，哪些描述正確？
   - (A) 可以模擬二維矩陣 (B) 修改子串列的內容會反映在父串列中 (C) 必須使用 `deepcopy` 才能完全獨立複製 (D) 子串列的長度必須相同
   - *答案*: A, B, C (KP5.4)

8. **Q18**: `sorted()` 函數的參數有哪些？
   - (A) iterable (B) key (C) reverse (D) inplace
   - *答案*: A, B, C (KP5.5)

9. **Q19**: 下列哪些可以用來檢查串列是否為空？
   - (A) `if not my_list:` (B) `if len(my_list) == 0:` (C) `if my_list == []:` (D) `if my_list is None:`
   - *答案*: A, B, C (KP5.1)

10. **Q20**: Tuple 的應用場景包括？
    - (A) 函數回傳多個值 (B) 儲存不應被修改的設定 (C) 作為字典的鍵 (D) 需要頻繁增減元素的動態陣列
    - *答案*: A, B, C (KP5.3)

### 3. 填充題 (Fill-in-the-blank) - 共 10 題

1. **Q21**: 想要在串列末尾加入一個元素，應使用 __________ 方法。
   - *答案*: append (KP5.1)

2. **Q22**: 串列生成式 `[x for x in range(10) if x % 2 != 0]` 會產生所有的 __________ (填奇數或偶數)。
   - *答案*: 奇數 (KP5.2)

3. **Q23**: Python 中將 Tuple 拆解並賦值給多個變數的過程稱為 __________ (Unpacking)。
   - *答案*: 解構賦值 (或 拆解) (KP5.3)

4. **Q24**: 若要進行「深拷貝」，需要匯入 __________ 模組。
   - *答案*: copy (KP5.4)

5. **Q25**: `L = [10, 20, 30]`, 執行 `L.pop(1)` 後，`L` 的內容為 __________。
   - *答案*: [10, 30] (KP5.1)

6. **Q26**: 元組 (Tuple) 使用的括號型式是 __________。
   - *答案*: 小括號 (或 ()) (KP5.3)

7. **Q27**: `fruits = ['apple', 'banana', 'cherry']`, 執行 `fruits.sort(key=len)` 後，第一個元素是 __________。
   - *答案*: apple (KP5.5)

8. **Q28**: 串列中計算特定元素出現次數的方法是 __________。
   - *答案*: count (KP5.1)

9. **Q29**: 在解構賦值中，可以使用星號 __________ 來代表剩餘的所有元素。
   - *答案*: * (KP5.3)

10. **Q30**: `[1] * 3` 的執行結果是 __________。
    - *答案*: [1, 1, 1] (KP5.1)
