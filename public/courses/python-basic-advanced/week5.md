## 一、 單元講解 (Lecture) - 總計 100 分鐘

### 1. 串列基礎與可變性 (Mutability) (20 min)
*   **串列 (List) 特性 (KP5.1)**：
    *   有序的集合，可以包含不同型別的物件。
    *   **可變性 (Mutable)**：可以直接修改元素內容、增加或刪除元素。
*   **記憶體配置**：串列儲存的是物件的引用 (Reference)。
*   **常用操作**：`.append()`, `.extend()`, `.insert()`, `.pop()`, `.remove()`。

### 2. 串列生成式 (List Comprehension) (20 min)
*   **語法結構 (KP5.2)**：`[expression for item in iterable if condition]`。
*   **優勢**：比傳統 `for` 迴圈更簡潔且執行效率通常更高。
*   **應用場景**：資料過濾、數值轉換、建立特定規律的串列。

### 3. 元組特性與解構賦值 (Tuple Unpacking) (20 min)
*   **元組 (Tuple) 特性 (KP5.3)**：
    *   **不可變性 (Immutable)**：一旦建立就不能修改內容。
    *   佔用記憶體較小，執行速度較快，常作為不可變的資料記錄。
*   **解構賦值 (Unpacking)**：
    *   `a, b = (10, 20)`。
    *   使用 `*` 收集多餘元素：`first, *middle, last = [1, 2, 3, 4, 5]`。

### 4. 巢狀結構與深淺拷貝 (20 min)
*   **巢狀串列 (Nesting) (KP5.4)**：串列中包含串列，用於表示矩陣或多維資料。
*   **淺拷貝 (Shallow Copy)**：`list.copy()` 或 `list[:]`，僅複製第一層引用。
*   **深拷貝 (Deep Copy)**：使用 `copy.deepcopy()`，遞迴複製所有層級的物件。

### 5. 排序與搜尋 (20 min)
*   **排序 (KP5.5)**：
    *   `.sort()`：就地排序 (In-place)，修改原串列。
    *   `sorted()`：回傳新串列，不影響原物件。
    *   **Key 參數**：使用 `key=len` 或 `key=str.lower` 進行自定義排序。
*   **搜尋與統計**：`in` 運算子、`.index()`, `.count()`。

## 二、 動手實作 (Lab) - 總計 50 分鐘

### 實作一：學生成績管理系統 (20 min)
**任務目標**：練習串列的基本操作與排序。
1.  建立一個串列 `grades = [85, 92, 78, 90, 88]`。
2.  新增一個學生成績 95 分。
3.  刪除最低分的成績。
4.  將成績由高到低排序並輸出。
5.  計算平均分數。

### 實作二：九九乘法表生成器 (15 min)
**任務目標**：練習巢狀串列與串列生成式。
1.  使用串列生成式建立一個 9x9 的巢狀串列，其中每個元素為 `i * j` 的結果。
2.  印出第三列 (3x1 到 3x9) 的結果。

### 實作三：座標互換與解構 (15 min)
**任務目標**：練習 Tuple 解構與應用。
1.  給定一個點的座標 `point = (10, 20, 30)`。
2.  使用解構賦值將 `x, y, z` 分別提取出來。
3.  寫一個交換函數的概念，利用 Tuple 特性一行程式碼交換 `a` 與 `b` 的值。
