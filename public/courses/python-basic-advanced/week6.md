## 一、 單元講解 (Lecture) - 總計 100 分鐘

### 1. 字典基礎與雜湊表 (Hash Tables) (20 min)
*   **字典 (Dictionary) 特性 (KP6.1)**：
    *   鍵值對 (Key-Value pairs) 結構。
    *   **雜湊表原理**：透過 Key 的雜湊值 (Hash) 實現極速查找 (O(1) 平均時間複雜度)。
*   **Key 的限制**：必須是不可變 (Hashable) 物件，如字串、數字、元組。
*   **常用操作**：`.get()`, `.keys()`, `.values()`, `.items()`。

### 2. 集合特性與運算 (20 min)
*   **集合 (Set) 特性 (KP6.2)**：
    *   無序且元素唯一（不重複）。
    *   同樣基於雜湊表實現。
*   **數學運算**：聯集 (`|`), 交集 (`&`), 差集 (`-`), 對稱差集 (`^`)。
*   **應用場景**：去重、快速成員檢查 (`in`)。

### 3. 字典與集合生成式 (20 min)
*   **字典生成式 (KP6.3)**：`{key_expr: value_expr for item in iterable}`。
*   **集合生成式**：`{expr for item in iterable}`。
*   **案例分析**：將兩個串列合併為字典、快速轉換資料型態。

### 4. 預設字典與進階操作 (20 min)
*   **`collections.defaultdict` (KP6.4)**：處理 Key 不存在時的預設值，避免 `KeyError`。
*   **`collections.Counter`**：快速統計元素出現次數。
*   **字典合併**：`|` 運算子 (Python 3.9+) 與 `.update()`。

### 5. 雜湊與不可變性 (Immutability) (20 min)
*   **Hashable 概念 (KP6.5)**：為什麼串列不能當字典的鍵？
*   **FrozenSet**：不可變的集合，可用於字典的鍵。
*   **字典有序性**：自 Python 3.7+ 起，字典會保持插入順序。

## 二、 動手實作 (Lab) - 總計 50 分鐘

### 實作一：文章單字頻率統計 (20 min)
**任務目標**：練習字典與 Counter 的應用。
1.  給定一段英文文本（字串）。
2.  將文本轉為小寫並移除標點符號。
3.  使用字典統計每個單字出現的次數。
4.  (進階) 使用 `collections.Counter` 找出出現頻率最高的前三個單字。

### 實作二：聯絡人資料庫 (15 min)
**任務目標**：練習字典的增刪改查。
1.  建立一個字典 `contacts`，Key 為姓名，Value 為電話。
2.  實作功能：新增聯絡人、查詢電話 (使用 `.get()` 處理不存在的情況)、更新電話、刪除聯絡人。
3.  印出所有聯絡人的姓名與電話。

### 實作三：數據清洗與比對 (15 min)
**任務目標**：練習集合運算。
1.  有兩個串列 A 與 B，包含一些重複的員工編號。
2.  任務：
    - 找出所有不重複的員工編號。
    - 找出同時出現在 A 與 B 的員工。
    - 找出只出現在 A 的員工。

## 四、 課後測驗題庫 (Quiz) - 30 分鐘

### 1. 單選題 (Single Choice) - 共 10 題

1. **Q1**: 下列哪一個不是定義字典的正確方式？
   - (A) `d = {}` (B) `d = dict()` (C) `d = {1, 2, 3}` (D) `d = {1: "a"}`
   - *答案*: C (KP6.1, KP6.2)

2. **Q2**: 字典中透過 Key 搜尋 Value 的平均時間複雜度是？
   - (A) O(1) (B) O(n) (C) O(log n) (D) O(n^2)
   - *答案*: A (KP6.1)

3. **Q3**: 集合 `s = {1, 2, 2, 3}` 執行後，`len(s)` 的結果是？
   - (A) 4 (B) 3 (C) 2 (D) 1
   - *答案*: B (KP6.2)

4. **Q4**: 下列哪種物件**不能**作為字典的 Key？
   - (A) "Name" (B) 100 (C) (1, 2) (D) [1, 2]
   - *答案*: D (KP6.5)

5. **Q5**: `d = {"a": 1}; print(d.get("b", 0))` 的輸出是？
   - (A) None (B) 0 (C) KeyError (D) 1
   - *答案*: B (KP6.1)

6. **Q6**: 集合運算中，`{1, 2} & {2, 3}` 的結果是？
   - (A) {1, 2, 3} (B) {2} (C) {1, 3} (D) {1}
   - *答案*: B (KP6.2)

7. **Q7**: 哪種生成式可以產生 `{0: 0, 1: 1, 2: 4}`？
   - (A) `{x: x**2 for x in range(3)}` (B) `{x**2 for x in range(3)}` (C) `[x: x**2 for x in range(3)]` (D) `{x, x**2 for x in range(3)}`
   - *答案*: A (KP6.3)

8. **Q8**: 使用哪個模組可以獲得 `defaultdict`？
   - (A) sys (B) os (C) collections (D) math
   - *答案*: C (KP6.4)

9. **Q9**: `d1 = {"a": 1}; d2 = {"b": 2}; d3 = d1 | d2` (Python 3.9+)，`d3` 是？
   - (A) {"a": 1} (B) {"b": 2} (C) {"a": 1, "b": 2} (D) 報錯
   - *答案*: C (KP6.4)

10. **Q10**: 如何刪除字典 `d` 中鍵為 "key" 的元素？
    - (A) `d.remove("key")` (B) `del d["key"]` (C) `d.discard("key")` (D) `d.delete("key")`
    - *答案*: B (KP6.1)

### 2. 多選題 (Multiple Choice) - 共 10 題

1. **Q11**: 關於集合 (Set) 的描述，哪些正確？
   - (A) 元素是無序的 (B) 元素可以重複 (C) 可以用來快速去重 (D) 使用 `{}` 定義 (空集合除外)
   - *答案*: A, C, D (KP6.2)

2. **Q12**: 哪些方法可以用於遍歷字典？
   - (A) .keys() (B) .values() (C) .items() (D) .list()
   - *答案*: A, B, C (KP6.1)

3. **Q13**: 關於 `defaultdict`，哪些正確？
   - (A) 可以避免 `KeyError` (B) 初始化時需要提供一個工廠函數 (如 `int`, `list`) (C) 它繼承自內建的 `dict` (D) 只能儲存字串
   - *答案*: A, B, C (KP6.4)

4. **Q14**: 下列哪些是集合的合法數學運算符？
   - (A) `|` (聯集) (B) `&` (交集) (C) `-` (差集) (D) `+` (相加)
   - *答案*: A, B, C (KP6.2)

5. **Q15**: 關於雜湊 (Hashable)，哪些正確？
   - (A) 不可變物件通常是 Hashable 的 (B) 可變物件 (如 List) 是 Unhashable 的 (C) 字典的 Value 必須是 Hashable 的 (D) `id()` 相同的物件雜湊值一定相同
   - *答案*: A, B (KP6.5)

6. **Q16**: 字典生成式的應用包括？
   - (A) 交換 Key 和 Value (B) 過濾符合條件的鍵值對 (C) 將 Tuple 串列轉為字典 (D) 執行數學運算
   - *答案*: A, B, C, D (KP6.3)

7. **Q17**: 關於 Python 字典的順序性，哪些正確？
   - (A) 在 3.7+ 版本保證插入順序 (B) 在所有 Python 版本都沒有順序 (C) 可以使用 `OrderedDict` 顯式保證順序 (D) 排序後會永久改變儲存順序
   - *答案*: A, C (KP6.1)

8. **Q18**: 如何清空一個集合 `s`？
   - (A) `s.clear()` (B) `s = set()` (C) `del s[:]` (D) `s.remove_all()`
   - *答案*: A, B (KP6.2)

9. **Q19**: `collections.Counter` 提供的功能有哪些？
   - (A) 統計頻率 (B) `most_common()` 方法 (C) 支持集合風格的加減運算 (D) 自動排序
   - *答案*: A, B, C (KP6.4)

10. **Q20**: 下列哪些可以用來建立空集合？
    - (A) `{}` (B) `set()` (C) `set([])` (D) `collections.set()`
    - *答案*: B, C (KP6.2)

### 3. 填充題 (Fill-in-the-blank) - 共 10 題

1. **Q21**: 字典是透過 __________ 表 (Hash Table) 來實現快速查找的。
   - *答案*: 雜湊 (KP6.1)

2. **Q22**: 想要取得字典中所有的鍵值對，應使用 __________ 方法。
   - *答案*: items() (KP6.1)

3. **Q23**: 集合運算符 `^` 代表的是 __________ 差集。
   - *答案*: 對稱 (KP6.2)

4. **Q24**: 空字典的定義是 `{}`，但空集合必須使用 __________ 來定義。
   - *答案*: set() (KP6.2)

5. **Q25**: 在 `defaultdict(list)` 中，若查詢一個不存在的鍵，會自動回傳一個空的 __________。
   - *答案*: 串列 (或 List) (KP6.4)

6. **Q26**: 想要檢查一個元素是否存在於集合中，應使用 __________ 運算子。
   - *答案*: in (KP6.2)

7. **Q27**: 字典生成式 `{x: x*2 for x in (1, 2)}` 的結果是 __________。
   - *答案*: {1: 2, 2: 4} (KP6.3)

8. **Q28**: 一個物件若要能作為字典的鍵，它必須是 __________ (Hashable) 的。
   - *答案*: 可雜湊 (KP6.5)

9. **Q29**: 使用 `d.pop("name")` 會移除鍵為 "name" 的元素並回傳其 __________。
   - *答案*: 值 (或 Value) (KP6.1)

10. **Q30**: `Counter("abac").most_common(1)` 的結果是 __________。
    - *答案*: [('a', 2)] (KP6.4)
