# 課程三：第 1 週 - 機器學習發展史與基本概念 (ML History & Basic Concepts)

本週將帶領學生進入機器學習的世界，從歷史脈絡出發，理解這門學科如何從早期的邏輯推理演進至現代的數據驅動模式。我們將定義機器學習的核心三要素，並初步接觸 Scikit-Learn 的開發環境。

---

## 一、 單元講解 (Lecture) - 100 分鐘

### 1. 機器學習的歷史脈絡
- **早期探索**：圖靈測試 (Turing Test) 與艾倫·圖靈的願景。
- **誕生與起伏**：1956 年達特茅斯會議、Rosenblatt 的感知器 (Perceptron)。
- **AI 寒冬**：第一次與第二次寒冬的成因（算力不足、符號主義的局限）。
- **現代復興**：數據量爆炸與硬體（GPU/TPU）的發展。

### 2. 機器學習的核心定義 (Tom Mitchell)
- **T (Task)**：任務，如圖像分類、垃圾郵件過濾。
- **E (Experience)**：經驗，即訓練數據。
- **P (Performance)**：效能指標，如準確率、召回率。
- **ML 本質**：程式從經驗 $E$ 中學習，在任務 $T$ 上提升效能指標 $P$。

### 3. 三大機器學習範式
- **監督式學習 (Supervised Learning)**：有標籤 (Label) 的學習，如分類與回歸。
- **非監督式學習 (Unsupervised Learning)**：無標籤學習，如聚類與降維。
- **強化學習 (Reinforcement Learning)**：透過獎勵機制進行試錯學習，如 AlphaGo。

### 4. 機器學習工作流 (Workflow)
- **資料處理**：收集、清理、特徵提取。
- **建模**：模型選擇與訓練。
- **評估與部屬**：測試集驗證、模型微調、上線監控。

### 5. Scikit-Learn (sklearn) 入門
- **Estimator API**：統一的 `fit()` 與 `predict()` 接口。
- **內建資料集**：`datasets` 模組 (Iris, Digits)。
- **資料切分**：`train_test_split` 的重要性與隨機種子。

## 二、 動手實作 (Lab) - 50 分鐘

### 任務：踏入 ML 的第一步
1.  **環境配置與庫導入**：
    - 安裝並導入 `numpy`, `pandas`, `matplotlib`, `sklearn`。
2.  **探索 Iris 資料集**：
    - 使用 `load_iris()` 載入數據，觀察 `data`, `target` 與 `feature_names`。
    - 使用 Pandas DataFrame 展示資料特徵。
3.  **基礎資料切分實踐**：
    - 使用 `train_test_split` 將數據按 8:2 比例切分為訓練集與測試集。
    - 討論並驗證為什麼不能在測試集上訓練模型。
4.  **體驗 Estimator**：
    - 快速呼叫一個基礎模型（如 `KNeighborsClassifier`）進行 `fit` 與 `score`。
