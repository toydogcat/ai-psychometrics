## 一、 單元講解 (Lecture) - 100 分鐘

### 1. 線性回歸的數學模型
- **模型形式**：$y = wx + b$（單變數）或 $y = w_0 + w_1x_1 + ... + w_nx_n$（多變數）。
- **向量化表示**：$y = Xw$，其中 $X$ 是特徵矩陣，$w$ 是權重向量。
- **目標**：尋找最佳的 $w$ 與 $b$，使預測值與真實值最接近。

### 2. 損失函數 (Loss Function)：均方誤差 (MSE)
- **MSE 定義**：$\frac{1}{n} \sum (y_i - \hat{y}_i)^2$。
- **幾何意義**：殘差平方和的平均，反映了模型擬合數據的總體誤差。
- **凸性 (Convexity)**：MSE 是一個凸函數，這保證了我們可以找到全域最小值。

### 3. 普通最小平方法 (OLS) 的閉式解
- **正規方程 (Normal Equation)**：$w = (X^T X)^{-1} X^T y$。
- **局限性**：矩陣求逆的計算複雜度高（$O(n^3)$），且當矩陣不可逆時失效。

### 4. 模型評估指標：$R^2$ 分數
- **決定係數 ($R^2$)**：反映模型解釋變異的能力，範圍通常在 0 到 1 之間。
- **解釋**：$R^2 = 0.8$ 代表模型解釋了資料中 80% 的變異。

### 5. 線性回歸的基礎假設
- **線性關係**：特徵與標籤呈線性。
- **獨立性**：樣本之間相互獨立。
- **同質方差性**：殘差的方差不隨 X 變化。
