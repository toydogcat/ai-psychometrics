# 課程三：第 12 週 - 奇異值分解 (SVD) 與矩陣分解應用 (SVD & Matrix Factorization)

SVD 被譽為線性代數的巔峰，也是機器學習中許多算法（如 PCA, LSA）的底層核心。本週我們將從數學幾何意義出發，探索如何將一個矩陣分解為三個特殊矩陣，並應用於影像壓縮與推薦系統。

---

## 一、 單元講解 (Lecture) - 100 分鐘

### 1. SVD 的數學定義
- **分解形式**：$A = U \Sigma V^T$。
- **$U$**：左奇異向量矩陣（樣本間的空間關係）。
- **$\Sigma$**：奇異值對角矩陣（特徵的重要性強度）。
- **$V^T$**：右奇異向量矩陣（特徵間的空間關係）。

### 2. SVD 的幾何意義
- **變換三部曲**：旋轉 (Rotation) -> 縮放 (Scaling) -> 旋轉 (Rotation)。
- **解釋**：任何線性變換都可以拆解為這三個基礎動作。

### 3. SVD 與 PCA 的關係
- **底層實現**：現代 PCA 多使用 SVD 來實現，而不需要顯式計算協方差矩陣 $X^T X$。
- **優勢**：SVD 在數值運算上更穩定，且能處理非方陣。

### 4. 矩陣低秩逼近 (Low-rank Approximation)
- **原理**：只保留最大的 $k$ 個奇異值，其餘設為 0。
- **應用：影像壓縮**：用極少的數據量還原影像的主體輪廓。

### 5. 推薦系統初步：矩陣補全
- **潛在因子 (Latent Factors)**：透過 SVD 分解「用戶-物品」評分矩陣，發現隱藏的偏好。
- **潛在語義分析 (LSA)**：在自然語言處理中降維詞頻矩陣。
