# 課程四：第 16 週 - GPT 家族演進：走向生成式 AI (GPT Family)

如果說 BERT 擅長「理解」語言，那麼 GPT 則開啟了「創造」語言的大門。本週我們將追蹤 OpenAI 從 GPT-1 到 GPT-3 的演進歷程，理解「規模即一切」的 scaling law，以及 RLHF 如何讓 AI 變得像人類一樣會聊天。

---

## 一、 單元講解 (Lecture) - 100 分鐘

### 1. GPT (Generative Pre-trained Transformer)
- **結構**：Transformer 解碼器 (Decoder-only)。
- **核心目標**：自回歸生成。根據前文預測下一個詞。
- **哲學**：單向注意力，專注於「生成」。

### 2. 從 1 到 3：參數規模的躍遷
- **GPT-1 (2018)**：驗證預訓練的可行性。
- **GPT-2 (2019)**：零樣本學習 (Zero-shot) 的萌芽，證明了不需要微調也能做任務。
- **GPT-3 (2020)**：1750 億參數。湧現 (Emergent Abilities) 現象：會寫代碼、會翻譯、會推理。

### 3. 指令微調 (Instruction Tuning) 與 ChatGPT
- **痛點**：原始模型只會續寫，不會回答問題。
- **方案**：在指令數據集上訓練模型聽從命令。

### 4. RLHF：人類反饋強化學習
- **三階段**：
    1. 監督式微調 (SFT)。
    2. 訓練獎勵模型 (RM)。
    3. PPO 強化學習優化。
- **作用**：對齊人類價值觀，讓 AI 更有用、更誠實、更無害。

### 5. 幻覺 (Hallucination) 與限制
- **機率本質**：AI 是在預測下一個詞，而非查閱事實。
- **根源**：缺乏真實世界的物理連接與事實檢索。

## 二、 動手實作 (Lab) - 50 分鐘

### 任務：呼叫與對話實戰
1.  **API 調用**：
    - 學習如何使用 Python 調用 OpenAI API (如 gpt-3.5-turbo)。
2.  **開源模型本地部署**：
    - 嘗試使用 Ollama 或 HuggingFace 載入 Llama 3。
3.  **零樣本與少樣本測試**：
    - 讓模型在沒有範例的情況下執行分類。
    - 提供 3 個範例 (Few-shot)，觀察效能提升。
4.  **對話歷史管理**：
    - 實現一個簡單的聊天循環，學習如何將先前的對話內容餵回模型。

## 三 island 知識點回顧 (KP)

- **[KP 4.16.1] 自回歸 (Autoregressive) 邏輯**：理解逐詞生成的數學基礎 $P(x_t | x_{<t})$。
- **[KP 4.16.2] Scaling Law (規模定律)**：明白算力、數據量與參數量對智能水平的決定性影響。
- **[KP 4.16.3] 零樣本/少樣本學習能力**：理解模型如何從海量知識中「內推」出新任務的解答。
- **[KP 4.16.4] 模型對齊 (Alignment)**：掌握 RLHF 如何將冷冰冰的概率分布轉化為溫度的對話。
- **[KP 4.16.5] 上下文窗口 (Context Window)**：理解模型「記憶長度」的物理限制及其代價。
