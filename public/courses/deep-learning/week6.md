# 課程四：第 6 週 - 深度卷積網路演進：從 LeNet 到 ResNet (Architecture Evolution)

自 2012 年 AlexNet 奪冠以來，CNN 架構經歷了爆炸式的演進。本週我們將梳理這段傳奇歷史，理解大師們如何透過精巧的結構設計，讓神經網路從幾層演進至上百層，並解決深層訓練中的核心矛盾。

---

## 一、 單元講解 (Lecture) - 100 分鐘

### 1. LeNet-5 (1998)：CNN 的原型
- **背景**：由 Yann LeCun 提出，用於支票手寫數字識別。
- **結構**：卷積 -> 下採樣 -> 卷積 -> 下採樣 -> 全連接。

### 2. AlexNet (2012)：深度學習的引爆點
- **突破**：使用 GPU 加速、ReLU 激活、Dropout、數據增強。
- **意義**：ImageNet 大賽以壓倒性優勢奪冠，開啟了深度學習時代。

### 3. VGG (2014)：深度與規整的藝術
- **思想**：全部使用 3x3 小卷積核。
- **結論**：多個小卷積核疊加比一個大卷積核更好（參數量更少、非線性更多）。

### 4. GoogLeNet (2014)：橫向擴展與 Inception
- **創新**：Inception 模組，在同一層執行多種尺寸卷積。
- **亮點**：引入 1x1 卷積進行降維（瓶頸層設計）。

### 5. ResNet (2015)：跳躍連接的魔法
- **問題**：網路越深，準確率反而下降（退化問題）。
- **方案**：殘差連接 (Skip Connection)，讓網路學習「殘差」而非全映射。
- **影響**：使訓練上千層的網絡成為可能。

## 二、 動手實作 (Lab) - 50 分鐘

### 任務：打造你的 ResNet Block
1.  **結構複刻**：
    - 在 PyTorch 中手動編寫 `BasicBlock` 類別，實現輸入與輸出的相加。
2.  **維度匹配處理**：
    - 學習如何使用 1x1 卷積調整 Shortcut 路徑的特徵圖尺寸與通道數。
3.  **模型比較實驗**：
    - 加載預訓練的 VGG16 與 ResNet18。
    - 對比兩者的參數量與推理速度（FPS）。
4.  **訓練退化觀察**：
    - 模擬一個沒有殘差連接的 20 層網絡，觀察其在訓練初期的 Loss 下降困難。

## 三 island 知識點回顧 (KP)

- **[KP 4.6.1] 小卷積核的優勢**：理解為什麼 $2 \times (3 \times 3)$ 等效於一個 $5 \times 5$ 但更優秀。
- **[KP 4.6.2] 1x1 卷積的妙用**：掌握其在跨通道資訊整合與參數量控制中的「降維」作用。
- **[KP 4.6.3] 殘差學習原理**：深刻理解 $H(x) = F(x) + x$ 為什麼能讓梯度流動更順暢。
- **[KP 4.6.4] 全域平均池化 (GAP)**：理解為什麼現代網絡傾向於棄用巨大的全連接層。
- **[KP 4.6.5] 網絡規整化趨勢**：觀察從手工設計特徵到結構組件模組化的演進。
