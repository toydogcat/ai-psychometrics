# 課程：微積分中 - 第 10 週 - 數列 (Sequences) 課後測驗詳解

本文件提供第 10 週課後測驗題庫的詳細分析與解答，重點在於強化對數列極限判定、羅必達法則應用以及單調收斂定理的理解。

## 一、 單選題 (Single Choice) 詳解

1. **Q1**: 數列 $a_n = \frac{n!}{n^n}$ 的極限。
   - **分析**：利用夾擠定理。$a_n = \frac{1 \cdot 2 \cdot 3 \cdots n}{n \cdot n \cdot n \cdots n} = \frac{1}{n} \left( \frac{2}{n} \cdot \frac{3}{n} \cdots \frac{n}{n} \right)$。
   - **過程**：因為括號內每一項都 $\le 1$，所以 $0 < a_n \le \frac{1}{n}$。當 $n \to \infty$ 時，$1/n \to 0$。
   - **答案**: (C) 0 (KP10.5)

2. **Q2**: 下列哪一個數列是單調遞增的？
   - **分析**：(A) $1/n$ 遞減；(B) $\cos n$ 震盪；(D) $(-1)^n$ 震盪。
   - **過程**：對於 (C) $a_n = \frac{n}{n+1} = 1 - \frac{1}{n+1}$。隨著 $n$ 增加，$1/(n+1)$ 減少，故 $1 - 1/(n+1)$ 增加。
   - **答案**: (C) (KP10.4)

3. **Q3**: 若 $\lim a_n = L > 0$，求 $\lim \ln(a_n)$。
   - **分析**：連續函數定理。因為 $f(x) = \ln x$ 在 $L > 0$ 處連續。
   - **答案**: (A) $\ln L$ (KP10.3)

4. **Q4**: $\lim \frac{5^n}{n!}$ 的值。
   - **分析**：階乘的增長速度遠大於指數函數。
   - **過程**：可考慮項比值 $\frac{a_{n+1}}{a_n} = \frac{5}{n+1}$，當 $n > 4$ 時比值 $< 1$，數列最終趨於 0。
   - **答案**: (B) 0 (KP10.3)

5. **Q5**: $a_n = (-1)^n \frac{n}{n+1}$ 的性質。
   - **分析**：這是一個交錯數列。
   - **過程**：絕對值項 $\frac{n}{n+1} \to 1$。因此項在 1 與 -1 附近震盪，不收斂。
   - **答案**: (C) 發散（震盪） (KP10.2)

## 二、 多選題 (Multiple Choice) 詳解

11. **Q11**: 關於數列的敘述。
    - (A) 正確，收斂必有界。
    - (B) 錯誤，例如 $a_n = (-1)^n$ 有界但發散。
    - (C) 正確，單調收斂定理。
    - (D) 正確，若數列收斂，所有子數列必收斂至同一極限。
    - **答案**: A, C, D (KP10.2, 10.4)

12. **Q12**: 下列哪些數列發散？
    - (A) 正確，震盪。
    - (B) 正確，趨向 $\infty$。
    - (C) 錯誤，幾何數列 $|r|<1$ 收斂至 0。
    - (D) 正確，值在 $1, 0, -1, 0$ 間震盪。
    - **答案**: A, B, D (KP10.2)

14. **Q14**: 哪些數列收斂至 0？
    - (A) 正確，階乘勝過指數。
    - (B) 正確，$\lim_{x \to \infty} \frac{\ln x}{x^2} = \lim \frac{1/x}{2x} = 0$。
    - (C) 錯誤，底數 $>1$ 發散。
    - (D) 正確，夾擠定理 $\left| \frac{\sin n}{n} \right| \le \frac{1}{n} \to 0$。
    - **答案**: A, B, D (KP10.3, 10.5)

## 三、 填充題 (Fill-in-the-blank) 詳解

19. **Q19**: $\arctan(\infty) = \pi/2$。
20. **Q20**: $\lim \frac{n^2}{e^n} = 0$ (羅必達兩次，或指數增長快於多項式)。
21. **Q21**: 0 (見 Q1 詳解)。
22. **Q22**: $\ln(n+1) - \ln(n) = \ln(\frac{n+1}{n}) = \ln(1 + 1/n) \to \ln(1) = 0$。
23. **Q23**: 1 (若 $r=1, a_n=1 \to 1$；若 $|r|<1, a_n \to 0$；若 $r \le -1$ 或 $r > 1$ 發散)。
24. **Q24**: 1 (若 $n=1, a_1 = \frac{1+(-1)^2}{2} = 1$；若 $n=2, a_2 = \frac{1+(-1)^3}{2} = 0$)。
25. **Q25**: 0 (夾擠定理)。
28. **Q28**: $\lim \sqrt[n]{3^n(1 + (2/3)^n)} = 3 \cdot \lim (1 + (2/3)^n)^{1/n} = 3 \cdot 1^0 = 3$。

30. **Q30**: 距離（或誤差）。
