## 一、 期末總複習 (Final Review)

### 1. 無窮級數重點回顧 (Weeks 1-13)
*   **收斂判定**：必須熟練區分正項級數與交錯級數。掌握比值測試 (Ratio Test) 在冪級數收斂半徑計算中的決定性地位。
*   **泰勒級數**：掌握五大基本展開式（$e^x, \sin x, \cos x, \frac{1}{1-x}, \tan^{-1} x$）。學會利用級數求極限與估算積分。
*   **歐拉公式**：理解複數與三角函數的聯繫，$e^{i\theta} = \cos \theta + i \sin \theta$。

### 2. 多元微積分基礎回顧 (Weeks 14-17)
*   **極限與連續**：掌握「兩路徑測試法」證明極限不存在。理解多元連續性的嚴格定義。
*   **偏導數與全微分**：理解偏導數的幾何意義（沿軸切線斜率）。掌握全微分 $dz = f_x dx + f_y dy$ 在誤差估計中的應用。
*   **切平面與可微性**：學會建立切平面方程式，並理解偏導連續是可微的充分條件。
*   **多元連鎖律**：學會使用樹狀圖 (Tree Diagrams) 處理多層次變數間的變化率傳遞。

## 三、 動手實作 (Lab) - 30 分鐘

### 實作：期末綜合繪圖與計算驗證
利用 Python 驗證一個複雜函數的泰勒逼近與切平面逼近。

```python
import numpy as np
import matplotlib.pyplot as plt
import sympy as sp

def final_verification():
    # 1. 符號驗證：泰勒展開
    x = sp.symbols('x')
    f_expr = sp.cos(x)
    taylor_4 = f_expr.series(x, 0, 6).removeO()
    print(f"cos(x) 的 4 階泰勒多項式: {taylor_4}")

    # 2. 數值驗證：多元函數偏導
    y = sp.symbols('y')
    g_expr = x**2 * y + sp.sin(x*y)
    gx = sp.diff(g_expr, x)
    gy = sp.diff(g_expr, y)
    print(f"g(x, y) = x^2*y + sin(xy) 的偏導 gx: {gx}")
    print(f"g(x, y) = x^2*y + sin(xy) 的偏導 gy: {gy}")

    # 3. 視覺化：二元函數圖形
    x_vals = np.linspace(-2, 2, 50)
    y_vals = np.linspace(-2, 2, 50)
    X, Y = np.meshgrid(x_vals, y_vals)
    Z = X**2 - Y**2 # 馬鞍面
    
    fig = plt.figure()
    ax = fig.add_subplot(111, projection='3d')
    ax.plot_surface(X, Y, Z, cmap='coolwarm')
    ax.set_title("Final Review: Saddle Surface z = x^2 - y^2")
    plt.show()

if __name__ == "__main__":
    final_verification()
```
