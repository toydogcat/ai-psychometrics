## 一、 單選題 (Single Choice) 解答與分析

1. **答案：D**
   - **分析**：辛普森法則的誤差項正比於 $f^{(4)}(x) \Delta x^4$，即與 $n^4$ 成反比（因為 $\Delta x \propto 1/n$）。(KP5.3)

2. **答案：C**
   - **分析**：辛普森法則是基於成對的子區間進行拋物線擬合，因此 $n$ 必須為偶數。(KP5.3)

3. **答案：C**
   - **分析**：根據 $p$-級數積分判別法，$\int_1^\infty \frac{1}{x^p} dx$ 在 $p > 1$ 時收斂。選項中只有 (C) 的 $p=2 > 1$。(KP5.4)

4. **答案：A**
   - **分析**：若 $f''(x) > 0$，函數圖形是凹向上的（凸函數），則連結兩端點的弦會在曲線上方，因此梯形法會高估積分值。(KP5.2)

5. **答案：B**
   - **分析**：對於凹向上的函數，中點處的切線位於曲線下方，因此中點法會低估積分值。(KP5.1)

6. **答案：B**
   - **分析**：當 $x \to 0^+$ 時，$\ln x \to -\infty$，函數在積分下限處不連續，屬於第二型瑕積分。(KP5.5)

7. **答案：C**
   - **分析**：$\int_1^\infty e^{-x} dx = \lim_{t \to \infty} [-e^{-x}]_1^t = \lim_{t \to \infty} (-e^{-t} + e^{-1}) = 1/e$。(KP5.4)

8. **答案：D**
   - **分析**：辛普森法對三次以下的多項式皆為精確解，代數精度最高。(KP5.3)

9. **答案：B**
   - **分析**：$\int_0^\infty \frac{1}{1+x^2} dx = [\arctan x]_0^\infty = \frac{\pi}{2} - 0 = \frac{\pi}{2}$。(KP5.4)

10. **答案：B**
    - **分析**：「收斂 + 發散 = 發散」是極限運算的基本性質。

## 二、 填充題 (Fill-in-the-blank) 解答與分析

11. **答案：2.75**
    - **分析**：$\Delta x = 0.5$。節點為 $0, 0.5, 1, 1.5, 2$。$f(x)=x^2$。$T_4 = \frac{0.5}{2} [0 + 2(0.25+1+2.25) + 4] = 0.25 \times [11] = 2.75$。

12. **答案：$1/\ln 2$**
    - **分析**：令 $u = \ln x, du = \frac{1}{x} dx$。積分變為 $\int_{\ln 2}^\infty \frac{1}{u^2} du = [-1/u]_{\ln 2}^\infty = 1/\ln 2$。

13. **答案：相等 (或精確值)**
    - **分析**：辛普森法則對次數 $\le 3$ 的多項式具有精確性。

14. **答案：發散**
    - **分析**：$\int_0^1 \frac{1}{x-1} dx = [\ln|x-1|]_0^1 = -\infty$。

15. **答案：-1**
    - **分析**：對於 $\int_0^1 x^p dx$，其反導函數為 $\frac{x^{p+1}}{p+1}$。若要在 $x=0$ 處收斂，需 $p+1 > 0 \implies p > -1$。

16. **答案：4, 2**
    - **分析**：辛普森係數模式為 $1, 4, 2, 4, \dots, 2, 4, 1$。

17. **答案：$\pi$**
    - **分析**：$[\arctan x]_{-\infty}^\infty = \frac{\pi}{2} - (-\frac{\pi}{2}) = \pi$。

18. **答案：2**
    - **分析**：$(10-0)/5 = 2$。

19. **答案：收斂**
    - **分析**：因為 $\frac{1}{x^2+1} < \frac{1}{x^2}$ 且 $\int_1^\infty \frac{1}{x^2} dx$ 收斂。

20. **答案：1**
    - **分析**：梯形法公式首尾項係數均為 1。

## 三、 計算與證明題解答

21. **解答**：
    當 $p \neq 1$ 時，$\int_1^t x^{-p} dx = [\frac{x^{1-p}}{1-p}]_1^t = \frac{t^{1-p} - 1}{1-p}$。
    若 $p > 1$，則 $1-p < 0$，當 $t \to \infty$ 時 $t^{1-p} \to 0$。
    極限為 $\frac{0-1}{1-p} = \frac{1}{p-1}$，為一有限值，故收斂。

22. **解答**：
    $\Delta x = 0.25$。節點為 $0, 0.25, 0.5, 0.75, 1$。
    $S_4 = \frac{0.25}{3} [e^0 + 4e^{0.25^2} + 2e^{0.5^2} + 4e^{0.75^2} + e^1]$。

23. **解答**：
    $\int_0^{\pi/2} \sec x dx = \lim_{t \to \pi/2^-} [\ln|\sec x + \tan x|]_0^t = \infty$。發散。

24. **解答**：
    在 $x=1$ 處不連續，拆為 $\int_0^1 + \int_1^3$。
    $= [3(x-1)^{1/3}]_0^1 + [3(x-1)^{1/3}]_1^3 = (0 - (-3)) + (3 \cdot 2^{1/3} - 0) = 3 + 3\sqrt[3]{2}$。

25. **解答**：
    因為 $0 \le \frac{\sin^2 x}{x^2} \le \frac{1}{x^2}$，且 $\int_1^\infty \frac{1}{x^2} dx$ 收斂。
    根據比較檢定法，原積分收斂。

26. **解答**：
    辛普森法的誤差依賴於四階導數 $f^{(4)}(x)$。三次多項式的四階導數恆為 0，故誤差項為 0。

27. **解答**：
    令 $u = \ln x$，積分為 $\int_1^\infty \frac{1}{u} du = [\ln u]_1^\infty = \infty$。發散。

28. **解答**：
    此為凸函數性質。梯形法用弦逼近，弦在曲線上方，故 $T_n$ 高估；中點法等價於用切線逼近，切線在曲線下方，故 $M_n$ 低估。

29. **解答**：
    $\lim_{t \to -\infty} [\frac{1}{2} e^{2x}]_t^0 = \frac{1}{2}(e^0 - 0) = 0.5$。

30. **解答**：
    因為 $\lim_{x \to \infty} f(x) = 1$，存在 $M > 0$ 使得當 $x > M$ 時 $f(x) > 0.5$。
    則 $\int_M^\infty f(x) dx \ge \int_M^\infty 0.5 dx = \infty$。故原積分發散。
