2. **(A)** 令 $u = \ln x, du = dx/x$. $\int_0^1 u du = 1/2$.
3. **(B)** $\frac{1}{1+(e^x)^2} \cdot e^x = \frac{e^x}{1+e^{2x}}$.
4. **(C)** $u=x, dv=\cos x dx \implies v=\sin x$.
5. **(C)** 形式為 $\sqrt{x^2-a^2}$.
6. **(B)** 分母包含不可約二次式。
7. **(D)** $E_S \propto n^{-4}$.
8. **(C)** $p=2 > 1$.
9. **(B)** $\pi \int_0^1 x dx = \pi/2$.
10. **(B)** $\sqrt{dx^2+dy^2}$.
11. **(B)** $\frac{1}{2} \int_1^3 x^2 dx = \frac{1}{2} [x^3/3]_1^3 = \frac{1}{2}(9 - 1/3) = 13/3$.
12. **(A)** $\int_1^\infty x^{-2} dx = [-1/x]_1^\infty = 1$.
13. **(D)** 壓力計算通常考慮相對壓力（錶壓力）。
14. **(C)** $4\pi r^2$.
15. **(A)** $\bar{x} = M_y / A$.

## 三、 計算與證明題 解答

31. **解答**：$I = \frac{1}{2} e^x (\sin x - \cos x) + C$（兩次分部積分）。
32. **解答**：$\frac{3x-1}{(x-2)(x+3)} = \frac{1}{x-2} + \frac{2}{x+3} \implies \ln|x-2| + 2\ln|x+3| + C$。
33. **解答**：$V = \pi \int_0^\pi \sin^2 x dx = \frac{\pi}{2} \int_0^\pi (1-\cos 2x) dx = \frac{\pi^2}{2}$。
34. **證明**：$\lim_{t \to \infty} [-e^{-x}]_0^t = \lim (1 - e^{-t}) = 1$。
35. **解答**：$A = 1/4$. $M_y = \int_0^1 x^4 dx = 1/5$. $M_x = \int_0^1 \frac{1}{2}x^6 dx = 1/14$. 形心 $(4/5, 2/7)$。
36. **解答**：$W = \int_0^1 \rho g (0.5) (1-y) dy = 0.5 \rho g [y - y^2/2]_0^1 = 0.25 \rho g = 2450$ J。
37. **解答**：令 $x=2\sin\theta$. $\int 4\cos^2\theta d\theta = 2(\theta + \sin\theta\cos\theta) = 2\sin^{-1}(x/2) + \frac{1}{2}x\sqrt{4-x^2} + C$。
38. **收斂**：因為 $\frac{2+\sin x}{x^2} \le \frac{3}{x^2}$ 且 $\int_1^\infty \frac{3}{x^2} dx$ 收斂。
39. **解答**：$\frac{1}{1-0} \int_0^1 \sqrt{1-x^2} dx = \pi/4$ (四分之一圓面積)。
40. **證明**：辛普森法的誤差依賴於四階導數。$x^3$ 的四階導數為 0，故誤差為 0，結果精確。
