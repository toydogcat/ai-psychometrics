## 三、 計算題解答

21. **解答**：
    利用表格法或兩次分部積分：$\int x^2 e^x dx = (x^2 - 2x + 2)e^x + C$。

22. **解答**：
    $[\sin^{-1} x]_0^1 = \pi/2$。

23. **解答**：
    交點 $x=0, 2$。在區間內 $2x \ge x^2$。
    $V = \pi \int_0^2 ((2x)^2 - (x^2)^2) dx = \pi [ \frac{4}{3}x^3 - \frac{1}{5}x^5 ]_0^2 = \pi (32/3 - 32/5) = 64\pi/15$。

24. **解答**：
    $\Delta x = 0.5$。節點 $0, 0.5, 1.0, 1.5, 2.0$。
    函數值 $1, 2/3, 1/2, 2/5, 1/3$。
    $S_4 = \frac{0.5}{3} [ 1 + 4(2/3) + 2(1/2) + 4(2/5) + 1/3 ] = \frac{1}{6} [ 1 + 8/3 + 1 + 8/5 + 1/3 ] = \frac{1}{6} [ 4 + 1.6 ] = 1.1$ (約略值)。

25. **解答**：
    當 $x \to \infty$，$\frac{x+1}{\sqrt{x^4-1}} \approx \frac{x}{x^2} = \frac{1}{x}$。
    與 $\int \frac{1}{x} dx$ 比較，可知發散。

26. **解答**：
    $y' = \frac{1}{3} \cdot \frac{3}{2}(x^2+2)^{1/2}(2x) = x\sqrt{x^2+2}$。
    $1+(y')^2 = 1 + x^2(x^2+2) = x^4 + 2x^2 + 1 = (x^2+1)^2$。
    $L = \int_0^1 (x^2+1) dx = [x^3/3 + x]_0^1 = 4/3$。

27. **解答**：
    $S = \int_0^\pi 2\pi \sin x \sqrt{1+\cos^2 x} dx$。

28. **解答**：
    $F = \int_0^2 \rho g y (2) dy = 2\rho g [y^2/2]_0^2 = 4\rho g = 39200$ N。

29. **解答**：
    $A = \int_0^4 \sqrt{x} dx = 16/3$。
    $M_y = \int_0^4 x^{3/2} dx = 64/5$。 $\bar{x} = (64/5)/(16/3) = 12/5 = 2.4$。
    $M_x = \int_0^4 \frac{1}{2}x dx = 4$。 $\bar{y} = 4/(16/3) = 3/4 = 0.75$。
    形心 $(2.4, 0.75)$。

30. **解答**：
    分部積分：$u=\ln x, dv=x^{-2}dx \implies du=1/x dx, v=-1/x$。
    $\int_1^\infty \frac{\ln x}{x^2} dx = \lim_{t \to \infty} ([-\frac{\ln x}{x}]_1^t + \int_1^t \frac{1}{x^2} dx) = 0 + [-1/x]_1^t = 1$。
