## 一、 單選題 (Single Choice) 詳細解析

### Q1：$\arcsin(1) = $？
*   **答案**：**(B)**
*   **解析**：找一個角度 $\theta \in [-\pi/2, \pi/2]$ 使得 $\sin \theta = 1$。故 $\theta = \pi/2$。

### Q2：$\frac{d}{dx}(\tan^{-1} x) = $？
*   **答案**：**(B)**
*   **解析**：這是基本公式 $\frac{d}{dx}(\arctan x) = \frac{1}{1+x^2}$。

### Q3：$\cosh 0 = $？
*   **答案**：**(B)**
*   **解析**：根據定義 $\cosh 0 = \frac{e^0 + e^{-0}}{2} = \frac{1+1}{2} = 1$。

### Q4：$\frac{d}{dx}(\cosh x) = $？
*   **答案**：**(A)**
*   **解析**：$\frac{d}{dx}(\frac{e^x+e^{-x}}{2}) = \frac{e^x-e^{-x}}{2} = \sinh x$。

### Q5：羅必達法則主要解決哪種問題？
*   **答案**：**(B)**
*   **解析**：羅必達法則是處理不定型極限的強大工具。

### Q6：$\lim_{x \to 0} \frac{\sin x}{x} = $？
*   **答案**：**(B)**
*   **解析**：使用羅必達法則：$\lim_{x \to 0} \frac{\cos x}{1} = \cos 0 = 1$。

### Q7：下列何者為 $\sinh x$ 的定義？
*   **答案**：**(B)**
*   **解析**：雙曲正弦定義為 $\frac{e^x-e^{-x}}{2}$。

### Q8：$\arctan(1) = $？
*   **答案**：**(A)**
*   **解析**：$\tan(\pi/4) = 1$ 且 $\pi/4 \in (-\pi/2, \pi/2)$。

### Q9：$\int \frac{1}{1+x^2} dx = $？
*   **答案**：**(B)**
*   **解析**：因為 $(\tan^{-1} x)' = \frac{1}{1+x^2}$，故積分為 $\tan^{-1} x + C$。

### Q10：$\cosh^2 x - \sinh^2 x = $？
*   **答案**：**(B)**
*   **解析**：這是雙曲函數最基本的恆等式，類比於 $\cos^2 \theta + \sin^2 \theta = 1$。

## 二、 多選題 (Multiple Choice) 詳細解析

### Q11：下列哪些是羅必達法則適用的直接不定型？
*   **答案**：**(A, B)**
*   **解析**：羅必達法則直接適用於 $0/0$ 與 $\infty/\infty$。其他形式（如 $0 \cdot \infty, 1^\infty$）需轉換後才能使用。

### Q12：關於 $\arcsin x$，哪些敘述正確？
*   **答案**：**(A, B, C, D)**
*   **解析**：(A) 定義域正確；(B) 值域正確；(C) $\sin^{-1}(-x) = -\sin^{-1} x$；(D) 導數 $1/\sqrt{1-x^2}$ 在 $(-1, 1)$ 恆正。

### Q13：關於雙曲函數，哪些正確？
*   **答案**：**(A, B, C, D)**
*   **解析**：(A) $\sinh(-x) = -\sinh x$；(B) $\cosh(-x) = \cosh x$；(C) 最小值為 $\cosh 0 = 1$；(D) 漸近於 $\pm 1$。

### Q14：下列導數正確的有？
*   **答案**：**(A, B, C, D)**
*   **解析**：這些皆為本週學習的核心導函數公式。注意 $\cos^{-1} x$ 帶負號，而 $\sinh x$ 的微分不帶負號。

### Q15：哪些極限可以用羅必達法則處理（可能需先變換）？
*   **答案**：**(A, B, C, D)**
*   **解析**：(A) $0 \cdot (-\infty)$；(B) $1^\infty$ 型（取對數）；(C) $\infty - \infty$ 型（通分）；(D) $0/0$ 型。

### Q16：下列恆等式正確的有？
*   **答案**：**(A, B, C, D)**
*   **解析**：這些是雙曲函數的基本運算性質與加法公式。

### Q17：反三角函數的值域限制包括：
*   **答案**：**(A, B, C)**
*   **解析**：(A, B, C) 皆正確。(D) $\text{arcsec}$ 的限制通常依教材而定，但 Stewart 常用 $[0, \pi/2) \cup [\pi, 3\pi/2)$。

### Q18：積分公式正確的有？
*   **答案**：**(A, B, C)**
*   **解析**：(D) 錯誤，$\int \sinh x dx = \cosh x + C$（無負號）。

### Q19：羅必達法則的先決條件包括：
*   **答案**：**(A, B, C)**
*   **解析**：(D) 錯誤，極限點可以是任何實數或 $\pm \infty$。

### Q20：下列哪些函數在 $x \to \infty$ 時比 $x^n$ 增長得快？
*   **答案**：**(A, C, D)**
*   **解析**：指數函數（包括 $e^x, 2^x, \sinh x$）的增長速度皆高於任何多項式函數。$\ln x$ 增長最慢。

## 三、 填充題 (Fill-in-the-blank) 詳細解析

### Q21：$\sin^{-1}(1/2) = \pi/6$
*   **解析**：$\sin(\pi/6) = 1/2$。

### Q22：$\frac{d}{dx}(\sin^{-1}(2x)) = \frac{2}{\sqrt{1-4x^2}}$
*   **解析**：連鎖律 $\frac{1}{\sqrt{1-(2x)^2}} \cdot 2$。

### Q23：$\lim_{x \to \infty} \frac{\ln x}{x} = 0$
*   **解析**：羅必達法則 $\lim_{x \to \infty} \frac{1/x}{1} = 0$。

### Q24：$\cosh x = (e^x+e^{-x})/2$
*   **解析**：基本定義。

### Q25：$\int \text{sech}^2(3x) dx = \frac{1}{3}\tanh(3x)+C$
*   **解析**：代換 $u=3x$。

### Q26：$\cos(\sin^{-1} x) = \sqrt{1-x^2}$
*   **解析**：令 $\theta = \sin^{-1} x \implies \sin \theta = x$，則 $\cos \theta = \sqrt{1-\sin^2 \theta} = \sqrt{1-x^2}$。

### Q27：$\lim_{x \to 0} \frac{e^x - 1}{x} = 1$
*   **解析**：羅必達法則或導數定義。

### Q28：$\tanh x$ 當 $x \to \infty$ 時趨近於 1
*   **解析**：$\lim_{x \to \infty} \frac{e^x-e^{-x}}{e^x+e^{-x}} = \lim_{x \to \infty} \frac{1-e^{-2x}}{1+e^{-2x}} = 1$。

### Q29：$\frac{d}{dx}(\sinh(x^2)) = 2x \cosh(x^2)$
*   **解析**：連鎖律。

### Q30：$dy/dx$ 在 $x=0$ 的值為 -1
*   **解析**：$(\cos^{-1} x)' = -1/\sqrt{1-x^2}$，代入 $x=0$ 得 $-1/1 = -1$。
