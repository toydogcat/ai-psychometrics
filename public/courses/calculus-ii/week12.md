# 課程：微積分中 - 第 12 週 - 泰勒與麥克勞林級數 (Taylor & Maclaurin Series)

本週將探討微積分中最壯觀的理論之一：如何將複雜的超越函數（如 $e^x, \sin x, \ln x$）表達為無窮多項式的形式。這不僅在理論上統一了多項式與超越函數，更是現代電腦進行科學計算、工程模擬以及工程近似的核心基礎。本週內容對應 **Stewart Calculus (Metric Edition) Chapter 11 Section 11.8 - 11.10**。

## 一、 單元講解 (Lecture) - 總計 100 分鐘

### 1. 冪級數與收斂半徑 (20 min) (KP12.1)
*   **課本對應**：Stewart Calculus Section 11.8.
*   **概念講解**：
    冪級數 (Power Series) 是形如 $\sum_{n=0}^{\infty} c_n (x-a)^n$ 的級數，其中 $a$ 是中心點。
    *   **收斂性**：對於一個冪級數，其收斂行為必為以下三者之一：
        1. 僅在 $x=a$ 處收斂。
        2. 對所有實數 $x$ 均收斂。
        3. 存在一個 $R > 0$（收斂半徑），使得當 $|x-a| < R$ 時絕對收斂，當 $|x-a| > R$ 時發散。
    *   **判定方法**：通常使用**比值檢定法 (Ratio Test)** 來求出 $R$。
*   **練習題與解答**：
    *   **練習題 12.1.1**：求冪級數 $\sum_{n=1}^{\infty} \frac{(x-3)^n}{n}$ 的收斂半徑與收斂區間。
    *   **解答**：
        1. 使用比值檢定法：$\lim_{n \to \infty} \left| \frac{(x-3)^{n+1}/(n+1)}{(x-3)^n/n} \right| = |x-3| \lim_{n \to \infty} \frac{n}{n+1} = |x-3|$。
        2. 令 $|x-3| < 1$，故收斂半徑 $R = 1$。
        3. 檢查端點：
           - $x=4$: $\sum 1/n$ 發散。
           - $x=2$: $\sum (-1)^n/n$ 收斂。
        4. 收斂區間為 $[2, 4)$。

### 2. 函數的冪級數表示法 (20 min) (KP12.2)
*   **課本對應**：Stewart Calculus Section 11.9.
*   **概念講解**：
    利用幾何級數 $\frac{1}{1-x} = \sum_{n=0}^{\infty} x^n$ ($|x|<1$) 作為母板，我們可以透過代換、微分或積分來推導其他函數的級數。
    *   **逐項微分與積分**：在收斂半徑內，冪級數可以像多項式一樣逐項進行微積分。
*   **練習題與解答**：
    *   **練習題 12.2.1**：求 $f(x) = \ln(1+x)$ 在 $x=0$ 處的冪級數表示法。
    *   **解答**：
        1. 我們知道 $\frac{d}{dx} \ln(1+x) = \frac{1}{1+x} = \frac{1}{1-(-x)}$。
        2. 當 $|x|<1$ 時，$\frac{1}{1-(-x)} = \sum_{n=0}^{\infty} (-x)^n = 1 - x + x^2 - x^3 + \dots$
        3. 兩邊積分：$\ln(1+x) = \int (1 - x + x^2 - \dots) dx = x - \frac{x^2}{2} + \frac{x^3}{3} - \dots + C$。
        4. 代入 $x=0$ 得 $C=0$。故 $\ln(1+x) = \sum_{n=1}^{\infty} (-1)^{n-1} \frac{x^n}{n}$。

### 3. 泰勒級數與麥克勞林級數的定義 (20 min) (KP12.3)
*   **課本對應**：Stewart Calculus Section 11.10.
*   **概念講解**：
    若函數 $f$ 在 $x=a$ 處具有各階導數，則 $f$ 在 $a$ 處的**泰勒級數 (Taylor Series)** 為：
    $$f(x) = \sum_{n=0}^{\infty} \frac{f^{(n)}(a)}{n!} (x-a)^n = f(a) + f'(a)(x-a) + \frac{f''(a)}{2!}(x-a)^2 + \dots$$
    當 $a=0$ 時，稱為**麥克勞林級數 (Maclaurin Series)**。
*   **練習題與解答**：
    *   **練習題 12.3.1**：求 $f(x) = e^x$ 在 $x=0$ 處的麥克勞林級數。
    *   **解答**：
        1. 因為 $f^{(n)}(x) = e^x$，故 $f^{(n)}(0) = e^0 = 1$ 對所有 $n$ 成立。
        2. 套用公式：$e^x = \sum_{n=0}^{\infty} \frac{1}{n!} x^n = 1 + x + \frac{x^2}{2!} + \frac{x^3}{3!} + \dots$

### 4. 泰勒多項式與餘項 (20 min) (KP12.4)
*   **課本對應**：Stewart Calculus Section 11.10.
*   **概念講解**：
    *   **$n$ 階泰勒多項式 $T_n(x)$**：泰勒級數的前 $n$ 項和。它是函數在 $x=a$ 附近的最佳 $n$ 次多項式逼近。
    *   **餘項 (Remainder) $R_n(x)$**：$R_n(x) = f(x) - T_n(x)$。
    *   **誤差估計**：若 $\lim_{n \to \infty} R_n(x) = 0$，則泰勒級數在該點確實收斂至 $f(x)$。

### 5. 泰勒定理及其證明 (20 min) (KP12.5)
*   **課本對應**：Stewart Calculus Section 11.10.
*   **概念講解**：
    **泰勒定理 (Taylor's Theorem)**：若 $f$ 在包含 $a$ 的區間 $I$ 上有 $n+1$ 階導數，則對 $I$ 中任意 $x$，存在 $z$ 介於 $x$ 與 $a$ 之間，使得：
    $$R_n(x) = \frac{f^{(n+1)}(z)}{(n+1)!} (x-a)^{n+1}$$
    這稱為**拉格朗日餘項 (Lagrange Form of the Remainder)**。
*   **證明概要**：
    利用廣義均值定理（Cauchy Mean Value Theorem）或反覆積分法。簡單直觀的證法是構造輔助函數並套用羅爾定理 (Rolle's Theorem)。
*   **練習題與解答**：
    *   **練習題 12.5.1**：估計用 $T_2(x) = 1 - \frac{x^2}{2}$ 近似 $\cos(0.1)$ 的最大可能誤差。
    *   **解答**：
        1. 餘項 $R_2(x) = \frac{f'''(z)}{3!} x^3 = \frac{\sin(z)}{6} (0.1)^3$。
        2. 因為 $|\sin(z)| \le 0.1$ (對於 $z \in [0, 0.1]$)，故誤差 $\le \frac{0.1 \cdot 0.001}{6} \approx 0.0000167$。

## 二、 動手實作 (Lab) - 30 分鐘

### 實作：泰勒多項式的動態逼近
我們將觀察隨著階數 $n$ 增加，$T_n(x)$ 如何逐漸貼合 $\sin(x)$。

```python
import numpy as np
import matplotlib.pyplot as plt
import math

def taylor_sin(x, n):
    """計算 sin(x) 的 n 階麥克勞林多項式"""
    val = 0
    for i in range(n + 1):
        term = ((-1)**i * x**(2*i + 1)) / math.factorial(2*i + 1)
        val += term
    return val

x_vals = np.linspace(-2*np.pi, 2*np.pi, 400)
y_sin = np.sin(x_vals)

plt.figure(figsize=(12, 6))
plt.plot(x_vals, y_sin, 'k', linewidth=2, label='True sin(x)')

for n in [0, 1, 2, 3]:
    y_approx = taylor_sin(x_vals, n)
    plt.plot(x_vals, y_approx, label=f'n={2*n+1} order')

plt.ylim(-2, 2)
plt.axhline(0, color='black', lw=1)
plt.axvline(0, color='black', lw=1)
plt.title('Taylor Polynomial Approximation of sin(x)')
plt.legend()
plt.grid(True)
plt.show()
```
