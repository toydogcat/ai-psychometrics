## 三、 計算與證明題解答

21. **解答**：
    考慮上半圓 $y = \sqrt{r^2 - x^2}$，繞 $x$-軸旋轉。
    $y' = \frac{-x}{\sqrt{r^2-x^2}} \implies 1+(y')^2 = \frac{r^2}{r^2-x^2}$。
    $S = \int_{-r}^r 2\pi \sqrt{r^2-x^2} \frac{r}{\sqrt{r^2-x^2}} dx = \int_{-r}^r 2\pi r dx = [2\pi r x]_{-r}^r = 4\pi r^2$。

22. **解答**：
    建立座標，水面 $y=20$（井口），水底 $y=0$。
    一層水的體積 $dV = \pi(1)^2 dy$。
    將這層水抽到井口需移動距離 $20-y$。
    $W = \int_0^{20} \rho g \pi (20-y) dy = 1000 \cdot 9.8 \cdot \pi [\int_0^{20} (20-y) dy] = 9800\pi [20y - \frac{1}{2}y^2]_0^{20} = 9800\pi (200) = 1,960,000\pi$ J。

23. **解答**：
    這是一個半徑為 2 的球體在 $x \in [-1, 1]$ 間的球台。
    $y' = -x/\sqrt{4-x^2}$。 $S = \int_{-1}^1 2\pi \sqrt{4-x^2} \frac{2}{\sqrt{4-x^2}} dx = \int_{-1}^1 8\pi dx = 16\pi$。

24. **解答**：
    建立座標，$y=0$ 為頂點，$y=3$ 為底邊（水面）。深度為 $3-y$。
    寬度函數 $w(y)$ 隨高度線性變化：$w(0)=0, w(3)=4 \implies w(y) = \frac{4}{3}y$。
    $F = \int_0^3 \rho g (3-y)(\frac{4}{3}y) dy = \frac{4\rho g}{3} [ \frac{3}{2}y^2 - \frac{1}{3}y^3 ]_0^3 = \frac{4\rho g}{3} (13.5 - 9) = 6\rho g$。
    若 $\rho g = 9800 \implies F = 58800$ N。

25. **解答**：
    1. $A = \int_0^1 (\sqrt{x} - x^2) dx = 2/3 - 1/3 = 1/3$。
    2. $M_y = \int_0^1 x(\sqrt{x} - x^2) dx = \int_0^1 (x^{3/2} - x^3) dx = [2/5 x^{5/2} - 1/4 x^4]_0^1 = 3/20$。
    3. $M_x = \int_0^1 \frac{1}{2}(x - x^4) dx = \frac{1}{2}[1/2 - 1/5] = 3/20$。
    形心為 $(M_y/A, M_x/A) = (9/20, 9/20) = (0.45, 0.45)$。

26. **解答**：
    曲線長 $L = \int ds$。形心 $y$-座標 $\bar{y} = \frac{1}{L} \int y ds$。
    旋轉路徑長 $2\pi \bar{y} = 2\pi \frac{1}{L} \int y ds$。
    $L \times (2\pi \bar{y}) = 2\pi \int y ds = S$。證畢。

27. **解答**：
    $\int_0^3 C x^2 dx = 1 \implies C [x^3/3]_0^3 = 1 \implies 9C = 1 \implies C = 1/9$。

28. **解答**：
    $A = \int_0^1 e^x dx = e-1$。
    $\bar{x} = \frac{1}{e-1} \int_0^1 x e^x dx = \frac{1}{e-1} [xe^x - e^x]_0^1 = \frac{1}{e-1}$。
    $\bar{y} = \frac{1}{e-1} \int_0^1 \frac{1}{2}e^{2x} dx = \frac{e^2-1}{4(e-1)} = \frac{e+1}{4}$。

29. **解答**：
    $W = \int_1^3 (3x^2 + 2x) dx = [x^3 + x^2]_1^3 = (27+9) - (1+1) = 34$ J。

30. **解答**：
    形心是區域質量的平均位置。對於非凸 (Non-convex) 區域（如 L 型板或圓環），重心可能位於空洞部分或物體外部。
    例如：圓環的形心位於圓心，但該處沒有實體物質。
