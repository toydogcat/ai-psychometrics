## 一、 單選題 (Single Choice) 詳細解析

### Q1：分部積分法公式為？
*   **答案**：**(B)**
*   **解析**：$\int u dv = uv - \int v du$。

### Q2：計算 $\int \ln x dx$ 時，$u$ 應選取？
*   **答案**：**(C)**
*   **解析**：根據 LIATE，對數函數優先選為 $u$。令 $u = \ln x, dv = dx$。

### Q3：使用 LIATE 法則，哪類函數在選取 $u$ 時優先度最高？
*   **答案**：**(C)**
*   **解析**：L (Logarithmic) > I (Inverse Trig) > A (Algebraic) > T (Trigonometric) > E (Exponential)。

### Q4：$\int x e^x dx = $？
*   **答案**：**(A)**
*   **解析**：$u=x, dv=e^x dx \implies uv - \int v du = x e^x - \int e^x dx = x e^x - e^x + C = (x-1)e^x + C$。

### Q5：表格法最適合處理哪種組合？
*   **答案**：**(B)**
*   **解析**：表格法適合多項式（次數有限，微分趨於 0）乘以指數或三角函數。

### Q6：計算 $\int \sin^2 x \cos^3 x dx$ 時，應令 $u = $？
*   **答案**：**(A)**
*   **解析**：因為 $\cos$ 次數為奇數，保留一個 $\cos x$，將 $\cos^2$ 化為 $1-\sin^2$，令 $u = \sin x$。

### Q7：$\int \sec^2 x dx = $？
*   **答案**：**(B)**
*   **解析**：這是基本導函數的反向運算：$(\tan x)' = \sec^2 x$。

### Q8：下列哪一個是 $\sin^2 x$ 的倍角公式？
*   **答案**：**(A)**
*   **解析**：$\sin^2 x = \frac{1-\cos 2x}{2}$。

### Q9：$\int \tan x dx = $？
*   **答案**：**(B)**
*   **解析**：$\int \frac{\sin x}{\cos x} dx = -\ln|\cos x| + C = \ln|\sec x| + C$。

### Q10：循環積分通常發生在什麼組合？
*   **答案**：**(B)**
*   **解析**：指數函數乘以三角函數（如 $e^x \sin x$）在兩次分部積分後會回到原式。

## 二、 多選題 (Multiple Choice) 詳細解析

### Q11：下列哪些積分通常使用分部積分法？
*   **答案**：**(A, B, C)**
*   **解析**：(A, B) 為典型的 LIATE 組合；(C) $\arctan x \cdot 1$；(D) $\int e^{x^2} dx$ 無初等函數解，不可用分部積分求得。

### Q12：關於三角積分 $\int \sin^m x \cos^n x dx$，正確敘述包括：
*   **答案**：**(A, B, C)**
*   **解析**：(A, B) 為奇數次方的提取策略；(C) 偶數次方需降次；(D) 錯誤，$u=\tan x$ 通常用於 $\sec$ 為偶數的情況。

### Q13：分部積分中 $v du$ 的部分：
*   **答案**：**(A, B, C)**
*   **解析**：(A) 這是分部積分的目的；(B) 如循環積分；(C) $v du$ 若無法積分則分部積分失敗；(D) 錯誤。

### Q14：下列積分結果正確的有：
*   **答案**：**(A, B, C, D)**
*   **解析**：皆為課本常見的重要積分公式與基本題型結果。

### Q15：表格法中：
*   **答案**：**(A, B, C)**
*   **解析**：(A, B, C) 皆為表格法正確操作步驟；(D) 錯誤，不限於指數。

### Q16：下列恆等式在三角積分中常用：
*   **答案**：**(A, B, C, D)**
*   **解析**：皆為三角函數降次與變換的核心公式。

### Q17：對於 $\int e^{2x} \cos 3x dx$：
*   **答案**：**(A, B, C)**
*   **解析**：(A, B) 指數與三角組合具循環性；(C) 選誰為 $u$ 只影響中間過程；(D) 表格法需以積分結尾，不能「直接」求出，需列方程。

### Q18：積分 $\int \tan^3 x \sec x dx$：
*   **答案**：**(A, B, C, D)**
*   **解析**：tan 次數為奇數時，提取 $\sec x \tan x$ 並令 $u = \sec x$ 是標準作法。

### Q19：關於 $\int \sec^3 x dx$：
*   **答案**：**(A, B, C, D)**
*   **解析**：這是三角積分中最著名的循環積分題型，常用於求曲線弧長。

### Q20：定積分的分部積分公式滿足：
*   **答案**：**(A, B, C, D)**
*   **解析**：定積分公式 $[uv]_a^b - \int_a^b v du$ 涵蓋了所有選項描述。

## 三、 填充題 (Fill-in-the-blank) 詳細解析

### Q21：$\int x \sin x dx = -x \cos x + \sin x + C$
*   **解析**：$u=x, dv=\sin x dx \implies -x \cos x - \int (-\cos x) dx$。

### Q22：符號序列為 +, -, +
*   **解析**：表格法正負交替。

### Q23：$\int \cos^2 x dx = \frac{x}{2} + \frac{\sin 2x}{4} + C$
*   **解析**：$\int \frac{1+\cos 2x}{2} dx = \frac{x}{2} + \frac{1}{2} \cdot \frac{\sin 2x}{2}$。

### Q24：可令 $u = \tan x$
*   **解析**：$\sec$ 次數為偶數，保留 $\sec^2$ 給 $du$。

### Q25：A 代表 Algebraic (代數)
*   **解析**：LIATE 縮寫。

### Q26：$\int \ln(2x) dx = x\ln(2x) - x + C$
*   **解析**：與 $\int \ln x dx$ 類似。

### Q27：$\int \sin x \cos x dx = -\frac{1}{4}\cos 2x + C$ (或 $\frac{1}{2}\sin^2 x$)
*   **解析**：$\int \frac{1}{2}\sin 2x dx = -\frac{1}{4}\cos 2x$。

### Q28：$\int \tan^2 x dx = \tan x - x + C$
*   **解析**：$\int (\sec^2 x - 1) dx$。

### Q29：$u = x^2$
*   **解析**：根據分部積分公式開頭部分。

### Q30：$\int_1^e \ln x dx = 1$
*   **解析**：$[x\ln x - x]_1^e = (e-e) - (0-1) = 1$。
