本文件提供第 6 週課後測驗題庫的詳細解答與重點分析。

## 一、 單選題 (Single Choice) 解答與分析

1. **答案：B**
   - **分析**：圓盤法體積元 $dV = \pi R^2 dx$，故包含 $f(x)^2$。(KP6.1)

2. **答案：A**
   - **分析**：剝殼法繞 $y$-軸旋轉時，半徑為 $x$，高度為 $f(x)$，故對 $x$ 積分。(KP6.3)

3. **答案：B**
   - **分析**：公式要求 $f'(x)$ 連續且在區間內可積，否則積分可能不存在。(KP6.4)

4. **答案：B**
   - **分析**：常數函數的平均值即為該常數。$\frac{1}{b-a} \int_a^b c dx = \frac{c(b-a)}{b-a} = c$。(KP6.5)

5. **答案：B**
   - **分析**：墊圈面積是外圓面積減內圓面積：$\pi R^2 - \pi r^2 = \pi(R^2 - r^2)$。(KP6.2)

6. **答案：C**
   - **分析**：$f_{avg} = \frac{1}{\pi-0} \int_0^\pi \sin x dx = \frac{1}{\pi} [-\cos x]_0^\pi = \frac{1}{\pi} (1 - (-1)) = 2/\pi$。(KP6.5)

7. **答案：A**
   - **分析**：區域邊界以 $x=g(y)$ 給出且繞 $y$-軸旋轉，直接用圓盤法（對 $y$ 積分）最快。

8. **答案：B**
   - **分析**：$2\pi x$ 是半徑為 $x$ 的圓周長。(KP6.3)

9. **答案：B**
   - **分析**：體積在物理上恆為正，若算得負值通常是設定了錯誤的積分次序或半徑公式。

10. **答案：C**
    - **分析**：$f(x)$ 是常數，則 $f'(x)=0$。積分式變為 $\int_1^3 1 dx = 2$，即長度為 2 的水平線段。

## 二、 填充題 (Fill-in-the-blank) 解答與分析

11. **答案：$\int_0^1 \pi x^6 dx$**
    - **分析**：$V = \int \pi (x^3)^2 dx$。

12. **答案：$x + 1$**
    - **分析**：到 $x = -1$ 的距離為 $x - (-1) = x + 1$。

13. **答案：4/3**
    - **分析**：$\frac{1}{2} \int_0^2 x^2 dx = \frac{1}{2} [x^3/3]_0^2 = \frac{8/3}{2} = 4/3$。

14. **答案：$\int_0^{\pi/4} \sec x dx$**
    - **分析**：$f'(x) = -\tan x \implies \sqrt{1+\tan^2 x} = \sqrt{\sec^2 x} = \sec x$。

15. **答案：至少存在一點**
    - **分析**：這是積分均值定理的核心結論。

16. **答案：$2 - f(x)$**
    - **分析**：點 $(x, f(x))$ 到直線 $y=2$ 的垂直距離。

17. **答案：以 $y$ 為自變數的函數 $x=g(y)$**
    - **分析**：當曲線對 $y$ 軸比較容易表達時。

18. **答案：$\pi$**
    - **分析**：半徑為 1 的圓周長一半。

19. **答案：$\frac{\pi}{2}(e^2 - 1)$**
    - **分析**：$\pi \int_0^1 (e^x)^2 dx = \pi \int_0^1 e^{2x} dx = \pi [\frac{1}{2}e^{2x}]_0^1 = \frac{\pi}{2}(e^2 - 1)$。

20. **答案：$\int_a^b s(x)^2 dx$**
    - **分析**：利用切片法，截面積為 $s(x)^2$。

## 三、 計算與證明題解答

21. **解答**：
    考慮上半圓 $y = \sqrt{r^2 - x^2}$，繞 $x$-軸旋轉。
    $V = \int_{-r}^r \pi (r^2 - x^2) dx = \pi [r^2 x - \frac{1}{3}x^3]_{-r}^r = \pi (r^3 - \frac{1}{3}r^3 - (-r^3 + \frac{1}{3}r^3)) = \pi (\frac{2}{3}r^3 + \frac{2}{3}r^3) = \frac{4}{3}\pi r^3$。

22. **解答**：
    交點 $(0,0)$ 與 $(1,1)$。在 $[0,1]$ 內 $\sqrt{x} \ge x$。
    $V = \int_0^1 2\pi x (\sqrt{x} - x) dx = 2\pi \int_0^1 (x^{3/2} - x^2) dx = 2\pi [\frac{2}{5}x^{5/2} - \frac{1}{3}x^3]_0^1 = 2\pi(\frac{2}{5} - \frac{1}{3}) = \frac{2\pi}{15}$。

23. **解答**：
    $y' = \frac{1}{2}x - \frac{1}{2x}$。
    $1+(y')^2 = 1 + (\frac{x}{2} - \frac{1}{2x})^2 = 1 + \frac{x^2}{4} - \frac{1}{2} + \frac{1}{4x^2} = \frac{x^2}{4} + \frac{1}{2} + \frac{1}{4x^2} = (\frac{x}{2} + \frac{1}{2x})^2$。
    $L = \int_1^2 (\frac{x}{2} + \frac{1}{2x}) dx = [\frac{x^2}{4} + \frac{1}{2}\ln x]_1^2 = (1 + \frac{1}{2}\ln 2) - (1/4 + 0) = \frac{3}{4} + \frac{1}{2}\ln 2$。

24. **解答**：
    截面面積 $A = \int_{-2}^2 (4 - x^2) dx = [4x - \frac{1}{3}x^3]_{-2}^2 = (8 - 8/3) - (-8 + 8/3) = 32/3$。
    總體積 $V = A \times \text{Length} = \frac{32}{3} \times 10 = \frac{320}{3}$ 立方公尺。

25. **解答**：
    $\frac{1}{2a} \int_{-a}^a f(x) dx = \frac{1}{2a} \cdot 2 \int_0^a f(x) dx = \frac{1}{a} \int_0^a f(x) dx$。成立。

26. **解答**：
    $f_{avg} = \frac{1}{\pi/2} \int_0^{\pi/2} \cos x dx = \frac{2}{\pi} [\sin x]_0^{\pi/2} = 2/\pi$。

27. **解答**：
    $V = \int_0^\pi 2\pi x \sin x dx$。
    分部積分：$u=x, dv=\sin x dx \implies du=dx, v=-\cos x$。
    $V = 2\pi ([-x\cos x]_0^\pi + \int_0^\pi \cos x dx) = 2\pi (\pi + 0) = 2\pi^2$。

28. **解答**：
    $y' = m$。$L = \int_{x_1}^{x_2} \sqrt{1+m^2} dx = (x_2-x_1)\sqrt{1+m^2}$。
    這等於 $\sqrt{(x_2-x_1)^2 + (m(x_2-x_1))^2} = \sqrt{\Delta x^2 + \Delta y^2}$。

29. **解答**：
    $x=y^2$ 與 $x=1$ 交於 $y=-1, 1$。半徑 $R = 1 - y^2$。
    $V = \int_{-1}^1 \pi (1-y^2)^2 dy = \pi \int_{-1}^1 (1-2y^2+y^4) dy = \pi [y - \frac{2}{3}y^3 + \frac{1}{5}y^5]_{-1}^1 = \frac{16\pi}{15}$。

30. **解答**：
    $f$ 的平均值為 0。根據積分均值定理，存在 $c$ 使得 $f(c) = f_{avg} = 0$。
