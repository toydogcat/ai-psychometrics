# 第 14 週：量化投資入門：金融數據獲取 (yfinance/tushare)

## 一、 單元講解 (Lecture)
本單元將開啟 Python 在量化投資領域的應用，學習如何獲取全球金融市場的數據並進行預處理。 (100 分鐘)

### 1. 量化交易與數據流 (20 min)
- **量化交易流程**: 數據獲取 -> 策略開發 -> 回測驗證 -> 實盤執行。
- **數據類型**: 行情數據 (OHLCV)、基本面數據、宏觀經濟數據、替代數據。
- **頻率概念**: 高頻、日線、週線、季報數據。

### 2. yfinance 獲取全球行情 (20 min)
- **基礎用法**: 下載股票 (如 AAPL, 2330.TW)、指數 (^GSPC) 與加密貨幣數據。
- **參數配置**: 設置時間範圍 (start, end) 與 間隔 (interval)。
- **Ticker 對象**: 獲取公司資訊、分紅記錄與拆股歷史。

### 3. Tushare 獲取 A 股數據 (20 min)
- **API 認證**: 了解 Tushare Pro 的 Token 獲取與積分機制。
- **數據接口**: 獲取股票列表、日線行情與財務指標。
- **限流處理**: 處理 API 調用頻率限制的策略。

### 4. 金融數據預處理 (20 min)
- **復權處理 (Adjusted Price)**: 理解為什麼回測必須使用調整收盤價 (Adj Close)。
- **處理缺失值**: 金融時間序列中特定的填充策略（如前向填充）。
- **數據格式統一**: 處理不同來源數據的時間時區與標識符差異。

### 5. 收益率計算與基礎統計 (20 min)
- **簡單收益率**: `(P_t - P_{t-1}) / P_{t-1}`。
- **對數收益率**: `log(P_t / P_{t-1})` 及其可累加性優勢。
- **基礎指標**: 計算均值 (期望收益) 與 波動率 (標準差)。

## 二、 動手實作 (Lab)
練習構建自己的金融數據庫。 (50 分鐘)

### 1. 標普 500 十年收益分析 (20 min)
- **任務**: 使用 `yfinance` 下載過去 10 年標普 500 指數數據。
- **要求**: 計算其年度累計收益率，並繪製收盤價趨勢圖。

### 2. 台股/美股龍頭對比 (15 min)
- **任務**: 同時獲取 台積電 (2330.TW) 與 蘋果 (AAPL) 的數據。
- **要求**: 將兩份數據按日期對齊，並計算兩者的相關性係數。

### 3. Tushare 基礎數據提取 (15 min)
- **任務**: 使用 Tushare 獲取某行業板塊內所有股票的最新估值 (PE/PB) 數據。
- **要求**: 找出該行業中 PE 最低的 5 支股票。
