# 第 10 週：科學計算基石：NumPy 數值運算

## 一、 單元講解 (Lecture)
本單元將學習 Python 數據分析的底層基石 NumPy，掌握高效的數值計算與矩陣運算。 (100 分鐘)

### 1. NumPy 基礎與 ndarray (20 min)
- **為什麼使用 NumPy**: 向量化運算 vs. Python 原始迴圈的性能差異。
- **ndarray 對象**: 維度 (ndim)、形狀 (shape) 與數據類型 (dtype)。
- **創建數組**: `np.array()`, `np.zeros()`, `np.ones()`, `np.arange()`, `np.linspace()`。

### 2. 數組索引與切片 (20 min)
- **基礎切片**: 一維與多維數組的子集選取。
- **花式索引 (Fancy Indexing)**: 使用整數數組進行索引。
- **布林索引**: 利用邏輯條件篩選數據。

### 3. 維度變換與合併 (20 min)
- **形狀調整**: `reshape()`, `flatten()`, `ravel()`。
- **轉置與軸對換**: `transpose()` 與 `.T` 屬性。
- **合併與拆分**: `vstack()`, `hstack()`, `concatenate()`。

### 4. 廣播機制 (Broadcasting) (20 min)
- **廣播原則**: 了解不同形狀數組進行運算時的自動擴張規則。
- **維度匹配**: 理解後緣維度 (trailing dimension) 的重要性。
- **實戰案例**: 數組與標量、數組與向量的運算。

### 5. 數學運算與統計函數 (20 min)
- **通用函數 (ufunc)**: `sin()`, `exp()`, `sqrt()` 等元素級運算。
- **統計聚合**: `sum()`, `mean()`, `std()`, `max()`, `min()` 及其 `axis` 參數的應用。
- **線性代數基礎**: 矩陣乘法 `dot()` 或 `@` 運算符。

## 二、 動手實作 (Lab)
練習使用 NumPy 進行高效的數據運算。 (50 分鐘)

### 1. 百萬級數據運算對比 (20 min)
- **任務**: 分別使用 Python 列表與 NumPy 數組計算 100 萬個隨機數的平方和，並對比執行時間。
- **要求**: 使用 `time` 模組記錄差異。

### 2. 矩陣運算實踐 (15 min)
- **任務**: 創建兩個 3x3 矩陣，計算它們的點積（矩陣乘法），並求出結果矩陣的特徵值。

### 3. 圖像灰度處理 (模擬) (15 min)
- **任務**: 將一個隨機生成的 (100, 100, 3) 三維數組（代表 RGB 圖片）轉換為 (100, 100) 的二維灰度數組。
- **要求**: 使用加權平均公式：`0.299R + 0.587G + 0.114B`。
