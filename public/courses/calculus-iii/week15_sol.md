# 課程：微積分下 - 第 15 週 - 旋度與散度 (詳解)

## 一、 單選題 (Single Choice)

1. **答案：B** (KP15.1)
   *   **解析**：旋度 $\nabla \times \mathbf{F}$ 是兩個向量的叉積運算，結果仍為向量。

2. **答案：B** (KP15.2)
   *   **解析**：散度正值代表該點有淨流量向外流出，物理上稱為「源」。

3. **答案：B** (KP15.3)
   *   **解析**：純量函數梯度的旋度恆為零向量，這反映了保守場的無旋性質。

4. **答案：A** (KP15.2)
   *   **解析**：散度 $\nabla \cdot \mathbf{F}$ 是向量算子與向量場的點積運算，結果為純量函數。

5. **答案：A** (KP15.2)
   *   **解析**：$\text{div } \mathbf{F} = \partial_x(y) + \partial_y(z) + \partial_z(x) = 0 + 0 + 0 = 0$。

6. **答案：B** (KP15.4)
   *   **解析**：拉普拉斯算子 $\nabla^2 f = \text{div}(\text{grad } f)$，結果是純量。

7. **答案：B** (KP15.3)
   *   **解析**：若區域簡單連通且旋度為零，則該場必為梯度場（保守場）。

8. **答案：B** (KP15.4)
   *   **解析**：旋度場的散度恆為零。

9. **答案：B** (KP15.2)
   *   **解析**：磁場的散度為零代表磁感線閉合，不存在磁單極子（磁荷）。

10. **答案：C** (KP15.1)
    *   **解析**：$\text{curl } \mathbf{F} = \langle 0, 0, \partial_x(x) - \partial_y(-y) \rangle = \langle 0, 0, 1 - (-1) \rangle = \langle 0, 0, 2 \rangle$，方向沿 $z$ 軸。

## 二、 多選題 (Multiple Choice)

11. **答案：B, C, D** (KP15.1)
    *   **解析**：(A) 旋度也可定義於二維（作為純量），三維是最常見形式。BCD 均符合旋度定義與性質。

12. **答案：A, B, D** (KP15.2)
    *   **解析**：(A) $0+0+0=0$ (B) $0+0+0=0$ (C) $1+1+1=3 \neq 0$ (D) $0+0+0=0$。

13. **答案：A, C** (KP15.5)
    *   **解析**：(A) 梯度是向量 (B) 散度是純量 (C) 旋度是向量 (D) 拉普拉斯作用於純量結果是純量。

14. **答案：A, B, C, D** (KP15.3, 15.4)
    *   **解析**：梯度場具備保守、路徑無關、無旋等所有列出的優良性質。

15. **答案：A, B, C, D** (KP15.5)
    *   **解析**：$\nabla$ 是算子向量，不滿足交換律（$\nabla \cdot \mathbf{F} \neq \mathbf{F} \cdot \nabla$），符合乘積法則。

16. **答案：A, B, C** (KP15.3)
    *   **解析**：(A) 重力場與 (B) 靜電場均為保守場，故無旋。(C) 均勻場梯度為零，故無旋。(D) 剛體旋轉流場旋度為 $2\boldsymbol{\omega}$，不為零。

17. **答案：A, B** (KP15.5)
    *   **解析**：此為向量恆等式 $\nabla \cdot (f\mathbf{F}) = f(\nabla \cdot \mathbf{F}) + \nabla f \cdot \mathbf{F}$。

18. **答案：A, B, D** (KP15.1, 15.2)
    *   **解析**：剛體旋轉流場是典型的有旋無散流場。

19. **答案：A, D** (KP15.2)
    *   **解析**：(A) $\partial_x x = 1$ (B) $1/3+1/3+1/3=1$（注意：若 B 指的是 $\langle x/3, y/3, z/3 \rangle$，則其散度也是 1。若題目為單選則需挑最直接的）。(D) $\partial_y y = 1$。

20. **答案：A, C** (KP15.4)
    *   **解析**：滿足拉普拉斯方程的函數稱為調和函數。

## 三、 填充題 (Fill-in-the-blank)

21. **3** (KP15.2)
    *   **解析**：$1 + 1 + 1 = 3$。
22. **$\mathbf{0}$** (KP15.1)
    *   **解析**：此場為 $\nabla(xy)$，保守場無旋。
23. **無散 (Solenoidal)** (KP15.2)
    *   **解析**：物理學術語。
24. **$\mathbf{i}, \mathbf{j}, \mathbf{k}$** (KP15.1)
    *   **解析**：行列式定義式。
25. **$\Delta$** (KP15.4)
    *   **解析**：常用數學符號。
26. **零 (Zero)** (KP15.3)
    *   **解析**：$\text{curl}(\text{grad } f) = \mathbf{0}$。
27. **$\nabla f \cdot \mathbf{F}$** (KP15.5)
    *   **解析**：乘積法則展開項。
28. **$z$ (或 $\mathbf{k}$)** (KP15.1)
    *   **解析**：二維場的旋度垂直於 $xy$ 平面。
29. **流出 (Diverging / Outflow)** (KP15.2)
    *   **解析**：散度的幾何解釋。
30. **0** (KP15.4)
    *   **解析**：$\text{div}(\text{curl } \mathbf{F}) = 0$。
