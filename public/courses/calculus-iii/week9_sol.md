## 二、 Python 加分題參考解答
```python
import sympy as sp

x, y = sp.symbols('x y')
f = x**2 + y**2
point = {x: 1, y: 1}

# 1. 計算梯度
grad_f = [sp.diff(f, var) for var in (x, y)]
grad_at_p = [float(g.subs(point)) for g in grad_f]
print(f"在 (1,1) 的梯度向量: {grad_at_p}")

# 2. 求等值線 (圓) 的切線方向
# x^2 + y^2 = 2 在 (1,1) 的隱函數求導: 2x + 2y(dy/dx) = 0
# dy/dx = -x/y. 在 (1,1) 處斜率為 -1
# 切向量方向可取為 [1, -1]
tangent_vec = [1, -1]

# 3. 檢查垂直 (點積是否為零)
dot_product = grad_at_p[0]*tangent_vec[0] + grad_at_p[1]*tangent_vec[1]
print(f"梯度與切向量的點積: {dot_product}")

if dot_product == 0:
    print("驗證成功：梯度垂直於等值線。")
```
