# 課程：微積分下 - 第 17 週 - Stokes 定理與 Divergence 定理 (詳解)

## 一、 單選題 (Single Choice)

1. **答案：A** (KP17.1)
   *   **解析**：Stokes 定理將三維閉曲線上的線積分轉換為以該曲線為邊界的曲面上的旋度面積分。

2. **答案：B** (KP17.3)
   *   **解析**：散度定理僅適用於封閉曲面（即圍成一個立體區域的曲面）。

3. **答案：C** (KP17.1)
   *   **解析**：在 $xy$ 平面上，單位法向量為 $\mathbf{k}$。

4. **答案：B** (KP17.3)
   *   **解析**：散度定理將閉曲面上的通量積分轉換為內部體積上的散度三重積分。

5. **答案：C** (KP17.3)
   *   **解析**：由散度定理，通量 $= \iiint (\text{div } \mathbf{F}) dV$。若 $\text{div } \mathbf{F} = 0$，則積分值恆為 0。

6. **答案：B** (KP17.1)
   *   **解析**：Stokes 定理的方向約定：當右手大拇指指向法向量方向時，四指彎曲方向即為邊界曲線的正向。

7. **答案：B** (KP17.4)
   *   **解析**：$\text{div } \langle x, 0, 0 \rangle = \partial_x(x) + \partial_y(0) + \partial_z(0) = 1$。

8. **答案：A** (KP17.2)
   *   **解析**：由散度定理，通量 $= \iiint \text{div}(\text{curl } \mathbf{A}) dV$。由於 $\text{div}(\text{curl } \mathbf{A})$ 恆為 0，故通量為 0。

9. **答案：B** (KP17.3)
   *   **解析**：散度定理在電磁學中常被稱為 Gauss 定理。

10. **答案：B** (KP17.5)
    *   **解析**：Green 定理是 Stokes 定理在 $xy$ 平面上的特殊情況。

## 二、 多選題 (Multiple Choice)

11. **答案：A, B, D** (KP17.1)
    *   **解析**：(C) 邊界曲線必須是閉合的，才能應用 Stokes 定理。

12. **答案：A, B, C** (KP17.4)
    *   **解析**：(D) 散度定理不直接用於求線積分。若 $\text{div } \mathbf{F}=1$，則三重積分結果即為體積值。

13. **答案：A, B** (KP17.3)
    *   **解析**：通量 $= \iiint 3 dV = 3 \times (\text{體積}) = 3 \times 10 = 30$。

14. **答案：A, B, C** (KP17.5)
    *   **解析**：廣義 Stokes 定理是現代微分幾何的核心，涵蓋了微積分中所有的基本定理。

15. **答案：A, B, C, D** (KP17.2)
    *   **解析**：(A)(B) 為等價條件。(C) 說明 $\iint (\text{curl } \mathbf{F}) \cdot \mathbf{n} dS = 0$。(D) 點的積分長度為 0。

16. **答案：A, B, C** (KP17.3)
    *   **解析**：區域可以是任何分段光滑閉曲面圍成的立體，不限於球體。

17. **答案：A, B, C** (KP17.5)
    *   **解析**：Green 定理可視為 Stokes 定理在二維空間的投影。

18. **答案：A, B, D** (KP17.3)
    *   **解析**：通量為 0 不代表場本身為 0（例如均勻流場穿過球體的通量為 0）。

19. **答案：A, B, C** (KP17.5)
    *   **解析**：分別對應梯度、散度與旋度算子。

20. **答案：A, B, C** (KP17.5)
    *   **解析**：(D) 線積分基本定理用於計算功或勢能，而非通量。

## 三、 填充題 (Fill-in-the-blank)

21. **$\text{curl } \mathbf{F}$ (或 $\nabla \times \mathbf{F}$)** (KP17.1)
    *   **解析**：Stokes 定理核心公式。
22. **$\text{div } \mathbf{F}$ (或 $\nabla \cdot \mathbf{F}$)** (KP17.3)
    *   **解析**：散度定理核心公式。
23. **體積 (Volume)** (KP17.4)
    *   **解析**：當被積函數為 1 時，積分結果為區域大小。
24. **單位法 (Unit normal)** (KP17.1)
    *   **解析**：幾何定義。
25. **3** (KP17.4)
    *   **解析**：$1+1+1=3$。
26. **相同 (Identical)** (KP17.2)
    *   **解析**：Stokes 定理指出積分值僅取決於邊界曲線。
27. **高斯 (Gauss)** (KP17.3)
    *   **解析**：常用別名。
28. **$4\pi/3$** (KP17.4)
    *   **解析**：立體幾何常識。
29. **2** (KP17.4)
    *   **解析**：$Q_x=1, P_y=-1 \implies 1 - (-1) = 2$。
30. **邊界 (Boundary)** (KP17.5)
    *   **解析**：所有向量積分定理的共同數學精神。
