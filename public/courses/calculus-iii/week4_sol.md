# 課程：微積分下 - 第 4 週 - 空間中的運動學 - 習題與測驗完整解答

本文件提供了第 4 週「空間中的運動學」課後測驗題庫共 30 題的完整解答與詳細剖析。

## 一、 單選題 (Single Choice) 詳細解析

1. **Q1**: 速率 $v(t)$ 是下列哪一項的長度？
   - **解析**: 速率定義為速度向量的模長，$v(t) = |\mathbf{v}(t)|$。
   - **答案**: (B)

2. **Q2**: 若加速度恆垂直於速度，則速率：
   - **解析**: 因為 $\mathbf{a} \cdot \mathbf{v} = 0$，則速率的變化率 $v' = (\mathbf{v} \cdot \mathbf{a})/v = 0$。故速率不變（例如勻速圓周運動）。
   - **答案**: (C)

3. **Q3**: 切線加速度 $a_T$ 的計算公式為？
   - **解析**: 切線加速度是速率對時間的變化率，$a_T = dv/dt = v'$。
   - **答案**: (A)

4. **Q4**: 副法向量 $\mathbf{B}$ 等於？
   - **解析**: 定義為切向量與法向量的外積，$\mathbf{B} = \mathbf{T} \times \mathbf{N}$。
   - **答案**: (B)

5. **Q5**: 拋體運動在忽略阻力時，哪個方向的加速度為 0？
   - **解析**: 重力僅作用於鉛直方向（$z$ 方向），水平方向（$x, y$）無力作用，故加速度為 0。
   - **答案**: (B)

6. **Q6**: 密切平面的法向量是？
   - **解析**: 密切平面由 $\mathbf{T, N}$ 張成，其法向量即為 $\mathbf{B}$。
   - **答案**: (C)

7. **Q7**: 若 $a_N = 0$，則質點的運動路徑為？
   - **解析**: $a_N = \kappa v^2 = 0 \implies \kappa = 0$（假設速度不為 0）。曲率為 0 的曲線是直線。
   - **答案**: (C)

8. **Q8**: 克卜勒第二定律（面積定律）實質上反映了什麼守恆？
   - **解析**: 有心力場（引力指向中心）作用下，角動量守恆，導致單位時間掃過面積不變。
   - **答案**: (C)

9. **Q9**: 法線加速度 $a_N$ 的物理意義是改變速度的？
   - **解析**: 切線分量改變速度大小，法線分量改變速度的方向。
   - **答案**: (B)

10. **Q10**: 單位切向量 $\mathbf{T}$ 對弧長 $s$ 的變化率大小 $|d\mathbf{T}/ds|$ 是？
    - **解析**: 根據曲率定義，$\kappa = |d\mathbf{T}/ds|$。
    - **答案**: (B)

## 二、 多選題 (Multiple Choice) 詳細解析

11. **Q11**: 下列哪些關於加速度分解的描述是正確的？
    - **解析**: (A, B, C) 均為加速度分解的標準公式。(D) 錯誤，$a_T$ 可為負（代表減速）。
    - **答案**: (A, B, C)

12. **Q12**: 關於 Frenet 標架 $\{\mathbf{T, N, B}\}$，下列哪些正確？
    - **解析**: (A, B, C, D) 均符合 Frenet 標架的性質：它是隨點移動的右手正交單位標架。
    - **答案**: (A, B, C, D)

13. **Q13**: 在拋體運動中，位置向量 $\mathbf{r}(t)$ 與哪些量有關？
    - **解析**: (A, B, C) 是積分常數與場強度。 (D) 質量在忽略阻力時不影響運動軌跡（伽利略原理）。
    - **答案**: (A, B, C)

14. **Q14**: 哪些運動狀態下，$a_T \neq 0$？
    - **解析**: (B, C, D) 均涉及速率的改變。 (A) 等速圓周運動速率不變，故 $a_T = 0$。
    - **答案**: (B, C, D)

15. **Q15**: 密切平面（Osculating Plane）的特點包括：
    - **解析**: (A, B, C, D) 均正確描述了密切平面的幾何屬性。
    - **答案**: (A, B, C, D)

16. **Q16**: 關於速率 $v$ 與曲率 $\kappa$ 對法線加速度 $a_N$ 的影響：
    - **解析**: (A, B, C) 正確。$a_N = \kappa v^2$ 恆正（或為 0），指向曲率中心。
    - **答案**: (A, B, C)

17. **Q17**: 質點在空間中做勻速運動（速度向量為常數），則：
    - **解析**: 速度不變則加速度為零向量，這暗示了路徑是直線（曲率 0）且速率不變。
    - **答案**: (A, B, C, D)

18. **Q18**: 向量 $\mathbf{B}$ 稱為副法向量，其性質有：
    - **解析**: (A, B, D) 是定義性質。(C) 錯誤，$d\mathbf{B}/ds$ 與 $\mathbf{N}$ 平行（其比例係數為扭率 $\tau$ 的負值）。
    - **答案**: (A, B, D)

19. **Q19**: 在研究行星運動時，我們通常假設：
    - **解析**: (A, B, C) 是標準模型的簡化。 (D) 錯誤，行星在橢圓軌道上速率不恆定。
    - **答案**: (A, B, C)

20. **Q20**: 下列哪些量是「標量 (Scalar)」？
    - **解析**: (A, B, C) 均為數值量。(D) 速度是向量。
    - **答案**: (A, B, C)

## 三、 填充題 (Fill-in-the-blank) 詳細解析

21. **Q21**: 若 $\mathbf{v} = \langle 3, 4, 0 \rangle$，則速率 $v = $ __________。
    - **解析**: $\sqrt{3^2+4^2} = 5$。
    - **答案**: 5

22. **Q22**: 加速度向量 $\mathbf{a}$ 在 $\mathbf{N}$ 方向的分量大小稱為 __________ 加速度。
    - **解析**: 垂直於運動方向的分量稱為法線加速度（或向心加速度）。
    - **答案**: 法線 (或 Normal)

23. **Q23**: 在圓周運動中，若半徑為 2，速率為 4，則法線加速度 $a_N = $ __________。
    - **解析**: $a_N = v^2/R = 16/2 = 8$。
    - **答案**: 8

24. **Q24**: 密切平面的方程式可以用點與法向量 __________ 來建立。
    - **解析**: 密切平面的法線方向即為副法向量 $\mathbf{B}$。
    - **答案**: B (或副法向量)

25. **Q25**: 拋體運動的水平軌跡在 $xy$ 平面上是一條 __________。
    - **解析**: 水平分量為 $x=v_{0x}t, y=v_{0y}t$，兩者成比例，故為直線。
    - **答案**: 直線

26. **Q26**: 若質點做直線運動，則其副法向量 $\mathbf{B}$ 是 __________。
    - **解析**: 直線運動中 $\mathbf{r}''$ 與 $\mathbf{r}'$ 平行，外積為 0，法向量無法確定。
    - **答案**: 不確定的

27. **Q27**: $a_T$ 的正負取決於速率是 __________ 還是減小。
    - **解析**: 若速率增加，$a_T > 0$；若速率減小，$a_T < 0$。
    - **答案**: 增加 (或 增大)

28. **Q28**: 單位切向量對時間的導數 $\mathbf{T}'(t)$ 的方向與向量 __________ 相同。
    - **解析**: $\mathbf{N} = \mathbf{T}'/|\mathbf{T}'|$，故方向與 $\mathbf{N}$ 相同。
    - **答案**: N (或單位法向量)

29. **Q29**: 克卜勒第一定律指出行星軌道形狀為 __________。
    - **解析**: 行星軌道為橢圓（Ellipse）。
    - **答案**: 橢圓

30. **Q30**: 加速度的長度平方滿足 $|\mathbf{a}|^2 = a_T^2 + $ __________。
    - **解析**: 由於 $\mathbf{T} \perp \mathbf{N}$，根據勾股定理，$|\mathbf{a}|^2 = a_T^2 + a_N^2$。
    - **答案**: $a_N^2$
