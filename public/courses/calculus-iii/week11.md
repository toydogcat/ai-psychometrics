## 一、 單元講解 (Lecture) - 總計 100 分鐘

### 1. 三重積分的定義與富比尼定理 (20 min) (KP11.1)
*   **概念講解**：
    三重積分是將函數 $f(x, y, z)$ 在三維空間區域 $E$ 上的累加。
    定義為：$\iiint_E f(x, y, z) \, dV = \lim_{\Delta \to 0} \sum f(x_i^*, y_i^*, z_i^*) \Delta V$。
*   **富比尼定理 (Fubini's Theorem)**：
    若 $E$ 是矩形盒 $[a, b] \times [c, d] \times [r, s]$，則三重積分可以寫成任何順序的逐次積分：
    $$\iiint_E f(x, y, z) \, dV = \int_a^b \int_c^d \int_r^s f(x, y, z) \, dz dy dx$$
*   **練習題**：
    *   **練習題 11.1.1**：計算 $\iiint_B xyz^2 \, dV$，其中 $B = [0, 1] \times [-1, 2] \times [0, 3]$。
    *   **解答**：
        $$\int_0^1 x \, dx \cdot \int_{-1}^2 y \, dy \cdot \int_0^3 z^2 \, dz = \left[\frac{1}{2}x^2\right]_0^1 \cdot \left[\frac{1}{2}y^2\right]_{-1}^2 \cdot \left[\frac{1}{3}z^3\right]_0^3 = \frac{1}{2} \cdot \left(2 - \frac{1}{2}\right) \cdot 9 = \frac{1}{2} \cdot \frac{3}{2} \cdot 9 = \frac{27}{4}$$

### 2. 一般區域上的三重積分 (20 min) (KP11.2)
*   **概念講解**：
    大多數情況下，區域 $E$ 不是盒子。我們通常將其分類為：
    *   **第 I 型區域**：$E = \{ (x, y, z) \mid (x, y) \in D, u_1(x, y) \le z \le u_2(x, y) \}$。
        $$\iiint_E f(x, y, z) \, dV = \iint_D \left[ \int_{u_1(x, y)}^{u_2(x, y)} f(x, y, z) \, dz \right] dA$$
*   **技巧**：先對「上下邊界」積分，再將問題簡化為 $xy$ 平面上的二重積分。
*   **練習題**：
    *   **練習題 11.2.1**：計算 $\iiint_E z \, dV$，其中 $E$ 是由四面體 $x+y+z=1$ 與三個坐標平面圍成的區域。
    *   **解答**：
        邊界為 $z=0$ 到 $z=1-x-y$。$xy$ 平面上的投影 $D$ 是由 $x=0, y=0, x+y=1$ 圍成的三角形。
        $$\int_0^1 \int_0^{1-x} \int_0^{1-x-y} z \, dz dy dx = \int_0^1 \int_0^{1-x} \frac{1}{2}(1-x-y)^2 dy dx$$
        令 $u = 1-x-y, du = -dy$：
        $$\int_0^1 \left[ -\frac{1}{6}(1-x-y)^3 \right]_0^{1-x} dx = \int_0^1 \frac{1}{6}(1-x)^3 dx = \left[ -\frac{1}{24}(1-x)^4 \right]_0^1 = \frac{1}{24}$$

### 3. 三重積分應用：體積 (20 min) (KP11.3)
*   **概念講解**：
    若令 $f(x, y, z) = 1$，則三重積分代表該區域的體積：
    $$V(E) = \iiint_E 1 \, dV$$
*   **物理意義**：體積是無數微小體積元素 $dV$ 的總和。
*   **練習題**：
    *   **練習題 11.3.1**：求由 $y=x^2, z=0, y+z=4$ 圍成實體的體積。
    *   **解答**：
        $z$ 範圍從 $0$ 到 $4-y$。$y$ 範圍從 $x^2$ 到 $4$。$x$ 範圍從 $-2$ 到 $2$。
        $$V = \int_{-2}^2 \int_{x^2}^4 \int_0^{4-y} dz dy dx = \int_{-2}^2 \int_{x^2}^4 (4-y) dy dx = \int_{-2}^2 [4y - \frac{1}{2}y^2]_{x^2}^4 dx$$
        $$= \int_{-2}^2 (16-8 - (4x^2 - \frac{1}{2}x^4)) dx = \int_{-2}^2 (8 - 4x^2 + \frac{1}{2}x^4) dx = [8x - \frac{4}{3}x^3 + \frac{1}{10}x^5]_{-2}^2 = \frac{256}{15}$$

### 4. 質量與質心 (20 min) (KP11.4)
*   **概念講解**：
    若物體密度為 $\rho(x, y, z)$：
    *   **質量 (Mass)**：$m = \iiint_E \rho(x, y, z) \, dV$。
    *   **對坐標平面的矩**：
        $M_{yz} = \iiint_E x \rho \, dV, \quad M_{xz} = \iiint_E y \rho \, dV, \quad M_{xy} = \iiint_E z \rho \, dV$。
    *   **質心**：$(\bar{x}, \bar{y}, \bar{z}) = (M_{yz}/m, M_{xz}/m, M_{xy}/m)$。
*   **練習題**：
    *   **練習題 11.4.1**：求密度均勻的單位球體第一卦限部分 ($x,y,z \ge 0$) 的質心。
    *   **提示**：由對稱性可知 $\bar{x} = \bar{y} = \bar{z}$。

---

### 5. 慣性矩 (Moments of Inertia) (20 min) (KP11.5)
*   **概念講解**：
    物體對坐標軸的慣性矩定義為：
    *   $I_x = \iiint_E (y^2 + z^2) \rho \, dV$
    *   $I_y = \iiint_E (x^2 + z^2) \rho \, dV$
    *   $I_z = \iiint_E (x^2 + y^2) \rho \, dV$
*   **物理意義**：慣性矩與質量分佈到軸的距離平方有關，決定了旋轉的難易程度。
*   **練習題**：
    *   **練習題 11.5.1**：計算邊長為 $L$、密度為 $\rho$ 的立方體對其中一條稜邊的慣性矩。

## 二、 動手實作 (Lab) - 總計 50 分鐘

### 實作：SciPy 三重積分 `tplquad` 計算
**任務目標**：利用 Python 計算複雜實體的體積與質量。

```python
import numpy as np
from scipy.integrate import tplquad

# 任務 1: 計算單位球體第一卦限的體積 (應為 (4/3*pi)/8 = pi/6)
# tplquad(func, x_low, x_up, y_low_func, y_up_func, z_low_func, z_up_func)
# 球方程: x^2 + y^2 + z^2 = 1 => z = sqrt(1 - x^2 - y^2)
def f_vol(z, y, x):
    return 1

vol, err = tplquad(f_vol, 0, 1, 
                   lambda x: 0, lambda x: np.sqrt(1 - x**2), 
                   lambda x, y: 0, lambda x, y: np.sqrt(1 - x**2 - y**2))

print(f"第一卦限體積: {vol:.6f} (理論值: {np.pi/6:.6f})")

# 任務 2: 計算該部分的質心 z 座標 (M_xy / m)
# mass = vol (因為 rho = 1)
def f_z_moment(z, y, x):
    return z

m_xy, err = tplquad(f_z_moment, 0, 1, 
                    lambda x: 0, lambda x: np.sqrt(1 - x**2), 
                    lambda x, y: 0, lambda x, y: np.sqrt(1 - x**2 - y**2))

z_bar = m_xy / vol
print(f"質心 z 座標: {z_bar:.6f} (理論值: 3/8 = 0.375)")
```
