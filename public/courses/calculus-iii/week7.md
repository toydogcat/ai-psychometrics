## 一、 單元講解 (Lecture)

### 1. 拉格朗日乘數法的核心思想 (KP7.1)
*   **概念講解**：
    假設我們要尋找目標函數 $f(x, y)$ 在約束條件 $g(x, y) = k$ 下的極值。
    *   **幾何直觀**：極值點發生在 $f$ 的等值線與約束曲線 $g = k$ **相切**的地方。
    *   **數學表達**：由於兩曲線相切，它們在該點的法向量（即梯度向量）必須**互相平行**。
    *   **核心方程**：
        $$\nabla f(x, y) = \lambda \nabla g(x, y)$$
        其中 $\lambda$ 稱為**拉格朗日乘數 (Lagrange Multiplier)**。
*   **證明概要**：
    設 $\mathbf{r}(t)$ 是約束曲線 $g(x, y) = k$ 的參數式。在極值點 $t_0$，複合函數 $h(t) = f(\mathbf{r}(t))$ 的導數為零：
    $h'(t_0) = \nabla f(\mathbf{r}(t_0)) \cdot \mathbf{r}'(t_0) = 0$。
    這說明 $\nabla f$ 垂直於切向量 $\mathbf{r}'$。又因為 $\nabla g$ 也垂直於 $\mathbf{r}'$，故 $\nabla f \parallel \nabla g$。

---

### 2. 單一約束條件的步驟 (KP7.2)
*   **操作步驟**：
    1. 列出方程組：
       $\begin{cases} \nabla f = \lambda \nabla g \\ g(x, y) = k \end{cases}$
    2. 展開為分量方程：
       $f_x = \lambda g_x$
       $f_y = \lambda g_y$
       $g(x, y) = k$
    3. 解出 $x, y, \lambda$。
    4. 計算所有解點的 $f$ 值，比較得出極值。
*   **練習題 7.2.1**：
    求 $f(x, y) = x^2 + y^2$ 在約束 $xy = 1$ 下的極小值。
    *   **解答**：
        1. $g(x, y) = xy = 1 \implies \nabla g = \langle y, x \rangle$。
        2. $\nabla f = \langle 2x, 2y \rangle$。
        3. 方程組：
           $2x = \lambda y$ (1)
           $2y = \lambda x$ (2)
           $xy = 1$ (3)
        4. (1) $\times x - (2) \times y \implies 2x^2 - 2y^2 = 0 \implies x^2 = y^2 \implies x = \pm y$。
        5. 代入 (3)：若 $x=y$，則 $x^2=1 \implies x=\pm 1$；若 $x=-y$，則 $-x^2=1$ (無實數解)。
        6. 點為 $(1, 1)$ 與 $(-1, -1)$。函數值 $f(1, 1) = f(-1, -1) = 2$。這即是極小值。

---

### 3. 兩個約束條件的拉格朗日法 (KP7.3)
*   **概念講解**：
    若有兩個約束 $g(x, y, z) = k$ 且 $h(x, y, z) = c$，則極值點滿足 $\nabla f$ 必須位於 $\nabla g$ 與 $\nabla h$ 所張成的平面內。
    *   **核心方程**：
        $$\nabla f = \lambda \nabla g + \mu \nabla h$$
*   **練習題 7.3.1**：
    求平面 $x+y+z=1$ 與柱面 $x^2+y^2=1$ 相交的交線上，離原點最遠與最近的點。
    *   **解答關鍵**：
        目標函數 $f = x^2 + y^2 + z^2$。約束 $g = x+y+z-1=0, h = x^2+y^2-1=0$。
        利用 $\nabla f = \lambda \nabla g + \mu \nabla h$ 解出。

---

### 4. 幾何應用：距離問題 (KP7.4)
*   **概念講解**：
    求點到曲面/曲線的最短距離是拉格朗日法的經典應用。
    *   **技巧**：最小化距離的平方 $d^2$ 通常比直接最小化 $d$ 更容易計算，且兩者極值點相同。
*   **練習題 7.4.1**：
    求點 $(1, 1, 1)$ 到平面 $x+2y+3z=6$ 的最短距離。
    *   **解答**：
        $f = (x-1)^2 + (y-1)^2 + (z-1)^2$，約束 $g = x+2y+3z-6=0$。
        $\nabla f = \langle 2(x-1), 2(y-1), 2(z-1) \rangle = \lambda \langle 1, 2, 3 \rangle$。
        解得 $x-1 = \lambda/2, y-1 = \lambda, z-1 = 3\lambda/2$。
        代入 $g=0$ 求出 $\lambda$，再得距離。

---

### 5. 經濟學應用：效用極大化 (KP7.5)
*   **概念講解**：
    消費者在預算限制下追求效用極大化。
    *   **模型**：極大化效用函數 $U(x, y)$，限制為預算線 $P_x x + P_y y = I$。
    *   **結論**：邊際效用之比等於價格之比（邊際替代率 = 價格比）。
        $$\frac{\partial U / \partial x}{P_x} = \frac{\partial U / \partial y}{P_y} = \lambda$$
        此 $\lambda$ 在經濟學中稱為「貨幣的邊際效用」。

## 二、 動手實作 (Lab) - 總計 50 分鐘

### 實作：利用 SymPy 解拉格朗日方程
**任務目標**：處理手算困難的非線性拉格朗日方程組。

```python
import sympy as sp

def solve_lagrange():
    x, y, z, lam = sp.symbols('x y z lam', real=True)
    
    # 定義目標函數與約束
    f = x * y * z  # 假設是求長方體最大體積
    g = 2*(x*y + y*z + z*x) - 12  # 約束：表面積固定為 12
    
    # 建立拉格朗日方程
    L = f - lam * g
    
    eq1 = sp.diff(L, x)
    eq2 = sp.diff(L, y)
    eq3 = sp.diff(L, z)
    eq4 = g  # 約束條件本身
    
    print("正在求解拉格朗日方程組...")
    solutions = sp.solve([eq1, eq2, eq3, eq4], (x, y, z, lam), dict=True)
    
    for sol in solutions:
        if all(val > 0 for val in [sol[x], sol[y], sol[z]]): # 只要正數解
            f_val = f.subs(sol)
            print(f"最優解: x={sol[x]}, y={sol[y]}, z={sol[z]}")
            print(f"最大目標值: {f_val}")

solve_lagrange()
```
