# 課程：微積分下 - 第 2 週 - 空間中的直線與平面 - 習題與測驗完整解答

本文件提供了第 2 週「空間中的直線與平面」課後測驗題庫共 30 題的完整解答與詳細剖析。

---

## 一、 單選題 (Single Choice) 詳細解析

1. **Q1**: 空間中通過兩點的直線有幾條？
   - **解析**: 幾何公理：通過不重合的兩點，有且只有一條直線。
   - **答案**: (B)

2. **Q2**: 平面 $2x - 3y + z = 4$ 的法向量為？
   - **解析**: 平面一般式 $ax+by+cz=d$ 的法向量即為係數構成的向量 $\langle a, b, c \rangle = \langle 2, -3, 1 \rangle$。
   - **答案**: (A)

3. **Q3**: 若兩平面的法向量垂直，則這兩個平面？
   - **解析**: 兩平面的夾角定義為其法向量的夾角。若法向量垂直（夾角 90 度），則平面亦垂直。
   - **答案**: (C)

4. **Q4**: 方程式 $x^2 + y^2 = 9$ 在三維空間中代表？
   - **解析**: 缺少變數 $z$，代表沿 $z$ 軸延伸。在 $xy$ 平面上是圓，故在 3D 中是圓柱面（Cylinder）。
   - **答案**: (C)

5. **Q5**: 直線 $\frac{x-1}{2} = \frac{y+2}{3} = z-4$ 的方向向量為？
   - **解析**: 對稱式分母即為方向向量分量。$z-4$ 可視為 $\frac{z-4}{1}$，故向量為 $\langle 2, 3, 1 \rangle$。
   - **答案**: (C)

6. **Q6**: 點 $(0, 0, 0)$ 到平面 $x+y+z=1$ 的距離為？
   - **解析**: $D = \frac{|0+0+0-1|}{\sqrt{1^2+1^2+1^2}} = \frac{1}{\sqrt{3}}$。
   - **答案**: (C)

7. **Q7**: 下列哪一個是橢球面的標準方程式？
   - **解析**: 橢球面所有變數均為二次項且係數皆為正，常數項為 1。(B) 符合 $\frac{x^2}{a^2} + \frac{y^2}{b^2} + \frac{z^2}{c^2} = 1$。
   - **答案**: (B)

8. **Q8**: 若直線與平面平行，則直線的方向向量 $\mathbf{v}$ 與平面的法向量 $\mathbf{n}$ 關係為？
   - **解析**: 直線平行於平面，代表直線方向垂直於平面的法向量，故 $\mathbf{v} \cdot \mathbf{n} = 0$。
   - **答案**: (A)

9. **Q9**: 兩平行平面 $x+y+z=1$ 與 $x+y+z=3$ 之間的距離為？
   - **解析**: $D = \frac{|1-3|}{\sqrt{1^2+1^2+1^2}} = \frac{2}{\sqrt{3}}$。
   - **答案**: (B)

10. **Q10**: 通過點 $(1, 0, 0)$ 且平行於 $yz$ 平面的平面方程式為？
    - **解析**: 平行於 $yz$ 平面代表其法向量為 $\mathbf{i} = \langle 1, 0, 0 \rangle$。方程式為 $1(x-1) = 0 \implies x=1$。
    - **答案**: (C)

## 二、 多選題 (Multiple Choice) 詳細解析

11. **Q11**: 關於直線方程式，下列哪些正確？
    - **解析**: (A, B, D) 皆為定義。 (C) 錯誤，若分母為零，對稱式必須改寫（例如若 $a=0$，則寫成 $x=x_0$ 與其餘兩項相等）。
    - **答案**: (A, B, D)

12. **Q12**: 哪些曲面的截面（Trace）包含橢圓？
    - **解析**: (A) 橢球面所有截面皆為橢圓。(B) 橢圓拋物面水平截面為橢圓。(C) 單葉雙曲面水平截面為橢圓。(D) 圓柱面水平截面為圓（圓是特殊的橢圓）。
    - **答案**: (A, B, C, D)

13. **Q13**: 平面 $ax + by + cz + d = 0$ 中，若 $d=0$ 則：
    - **解析**: (A) 代入 $(0,0,0)$ 滿足方程，故過原點。(C) 法向量始終為 $\langle a, b, c \rangle$。
    - **答案**: (A, C)

14. **Q14**: 判定兩直線是否為「歪斜線 (Skew lines)」的條件包括：
    - **解析**: 歪斜線定義為不同在任何平面上的兩直線。其特徵是不平行且不相交。
    - **答案**: (A, B, C)

15. **Q15**: 直線 $L_1$ 與 $L_2$ 平行的條件是其方向向量 $\mathbf{v}_1, \mathbf{v}_2$：
    - **解析**: (A) 互為倍數。(C) 外積為零向量。(D) 夾角為 0 或 180 度。
    - **答案**: (A, C, D)

16. **Q16**: 下列哪些是二次曲面？
    - **解析**: (A, B, C) 皆為二次方程定義。 (D) 平面是一次方程，不屬於二次曲面。
    - **答案**: (A, B, C)

17. **Q17**: 關於點到平面距離公式 $D$，下列敘述正確的是：
    - **解析**: (A) 分母 $\sqrt{a^2+b^2+c^2}$。(B) 分子是絕對值。(C, D) 距離必 $\ge 0$。
    - **答案**: (A, B, C, D)

18. **Q18**: 若平面 $\pi$ 通過點 $P, Q, R$，其法向量可以透過哪些運算得到？
    - **解析**: 法向量需同時垂直於平面內兩不平行向量。向量外積結果垂直於兩者。
    - **答案**: (A, B)

19. **Q19**: 二次曲面 $z = y^2 - x^2$ 的名稱是：
    - **解析**: 一個變數是一次，另外兩個是二次且符號相反，稱為雙曲拋物面（馬鞍面）。
    - **答案**: (A, B)

20. **Q20**: 直線與平面的關係可能為：
    - **解析**: 直線與平面最多交於一點、或平行、或直線在平面上（無限交點）。不可能恰好交於兩點但不重合。
    - **答案**: (A, B, C)

## 三、 填充題 (Fill-in-the-blank) 詳細解析

21. **Q21**: 通過 $(0,0,0)$ 且方向向量為 $\langle 1, 1, 1 \rangle$ 的直線參數式為 $x=t, y=t, z=$ __________。
    - **解析**: $z = 0 + 1t = t$。
    - **答案**: $t$

22. **Q22**: 平面 $x - 2y + 2z = 6$ 到原點的距離為 __________。
    - **解析**: $D = \frac{|0-0+0-6|}{\sqrt{1^2+(-2)^2+2^2}} = \frac{6}{3} = 2$。
    - **答案**: 2

23. **Q23**: 若兩平面平行，則它們的 __________ 向量必平行。
    - **解析**: 平面的朝向由法向量決定，平行平面具有平行的法向量。
    - **答案**: 法 (或 Normal)

24. **Q24**: 曲面 $z = x^2 + y^2$ 在 $z=4$ 處的截面是一個半徑為 __________ 的圓。
    - **解析**: $4 = x^2 + y^2 \implies r^2 = 4 \implies r = 2$。
    - **答案**: 2

25. **Q25**: 直線 $x=t, y=t, z=t$ 與平面 $x+y+z=3$ 的交點為 __________。
    - **解析**: 代入得 $t+t+t=3 \implies 3t=3 \implies t=1$。交點為 $(1, 1, 1)$。
    - **答案**: $(1, 1, 1)$

26. **Q26**: 空間中通過點 $(1, 1, 1)$ 且垂直於 $z$ 軸的平面方程式為 __________。
    - **解析**: 垂直於 $z$ 軸代表平行於 $xy$ 平面，法向量為 $\mathbf{k} = \langle 0, 0, 1 \rangle$。$z=1$。
    - **答案**: $z=1$

27. **Q27**: 單葉雙曲面 $\frac{x^2}{a^2} + \frac{y^2}{b^2} - \frac{z^2}{c^2} = 1$ 的對稱中心在 __________。
    - **解析**: 標準式中心位於原點 $(0, 0, 0)$。
    - **答案**: 原點 (或 $(0, 0, 0)$)

28. **Q28**: 方程式 $y = x^2$ 在 3D 空間中代表 __________ 面。
    - **解析**: 在 $xy$ 平面是拋物線，沿 $z$ 軸拉伸形成拋物柱面。
    - **答案**: 拋物柱 (或 Parabolic Cylinder)

29. **Q29**: 兩平面 $x+y=1$ 與 $x+y=2$ 的關係是 __________。
    - **解析**: 法向量皆為 $\langle 1, 1, 0 \rangle$，係數成比例但常數項比例不同，故平行。
    - **答案**: 平行

30. **Q30**: 點到平面距離公式中，法向量不需要是單位向量，但分母必須除以其 __________。
    - **解析**: 公式分母為 $\sqrt{a^2+b^2+c^2}$，即法向量的長度。
    - **答案**: 長度 (或 Magnitude)
