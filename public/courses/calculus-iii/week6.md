## 一、 單元講解 (Lecture)

### 1. 局部極值的定義與費馬定理 (KP6.1, KP6.2)
*   **概念講解**：
    *   **定義**：若點 $(a, b)$ 鄰域內所有點 $(x, y)$ 均滿足 $f(x, y) \leq f(a, b)$，則 $f(a, b)$ 是**局部極大值**。反之則為**局部極小值**。
    *   **費馬定理 (Fermat's Theorem for Functions of Two Variables)**：若 $f$ 在 $(a, b)$ 有局部極值，且其偏導數存在，則 $f_x(a, b) = 0$ 且 $f_y(a, b) = 0$。
    *   **臨界點 (Critical Point)**：滿足 $\nabla f(a, b) = \mathbf{0}$（即 $f_x=0, f_y=0$）或偏導數不存在的點。
*   **練習題 6.2.1**：
    求 $f(x, y) = x^2 + y^2 - 2x - 6y + 14$ 的臨界點。
    *   **解答**：
        1. $f_x = 2x - 2 = 0 \implies x = 1$。
        2. $f_y = 2y - 6 = 0 \implies y = 3$。
        3. 臨界點為 $(1, 3)$。配方後得 $f(x, y) = (x-1)^2 + (y-3)^2 + 4$，顯然在 $(1, 3)$ 有極小值 4。

---

### 2. 二階導數測試 (Second Derivatives Test) (KP6.3)
*   **定理與證明概要**：
    假設 $(a, b)$ 是 $f$ 的臨界點，且二階偏導連續。令判別式 (Hessian determinant) $D$ 為：
    $$D = D(a, b) = f_{xx}(a, b)f_{yy}(a, b) - [f_{xy}(a, b)]^2$$
    *   **情況 A**：若 $D > 0$ 且 $f_{xx}(a, b) > 0$，則 $f(a, b)$ 是**局部極小值**。
    *   **情況 B**：若 $D > 0$ 且 $f_{xx}(a, b) < 0$，則 $f(a, b)$ 是**局部極大值**。
    *   **情況 C**：若 $D < 0$，則 $(a, b)$ 是**鞍點 (Saddle Point)**。
    *   **情況 D**：若 $D = 0$，此測試**無結論**。
*   **練習題 6.3.1**：
    判定 $f(x, y) = x^4 + y^4 - 4xy + 1$ 的極值點。
    *   **解答**：
        1. 找臨界點：$f_x = 4x^3 - 4y = 0 \implies y = x^3$；$f_y = 4y^3 - 4x = 0 \implies x = y^3$。
           代入得 $x = (x^3)^3 = x^9 \implies x(x^8 - 1) = 0 \implies x = 0, 1, -1$。
           臨界點為 $(0, 0), (1, 1), (-1, -1)$。
        2. 計算二階導：$f_{xx} = 12x^2, f_{yy} = 12y^2, f_{xy} = -4$。
           $D = (12x^2)(12y^2) - (-4)^2 = 144x^2 y^2 - 16$。
        3. 測試：
           - $(0, 0): D = -16 < 0 \implies$ **鞍點**。
           - $(1, 1): D = 144-16 = 128 > 0, f_{xx} = 12 > 0 \implies$ **局部極小**。
           - $(-1, -1): D = 128 > 0, f_{xx} = 12 > 0 \implies$ **局部極小**。

---

### 3. 鞍點 (Saddle Points) (KP6.4)
*   **概念講解**：
    鞍點是臨界點，但不是局部極值。在某些方向看是極大值，某些方向看是極小值。
    *   **經典例子**：$f(x, y) = y^2 - x^2$。在原點 $(0, 0)$，$f_x = f_y = 0$。
        沿 $x$ 軸看（$y=0$）：$f = -x^2$，在 0 點是極大。
        沿 $y$ 軸看（$x=0$）：$f = y^2$，在 0 點是極小。
        其形狀像馬鞍，故名。
*   **練習題 6.4.1**：
    參考圖 `img/week6_kp6_4.webp`，解釋為何 $D < 0$ 代表鞍點。
    *   **解釋**：$D = f_{xx}f_{yy} - f_{xy}^2 < 0$ 意味著在該點曲面的曲率在不同方向上有相反的正負號，導致函數在某些方向上升而在其他方向下降。

### 4. 絕對極值與極值定理 (KP6.5)
*   **概念講解**：
    **極值定理 (Extreme Value Theorem)**：若 $f$ 在閉合且有界 (Closed and Bounded) 的區域 $D$ 上連續，則 $f$ 一定能取得絕對最大值與絕對最小值。
    *   **求法步驟**：
        1. 找出區域 $D$ 內部的所有臨界點，計算函數值。
        2. 找出區域 $D$ 邊界上的極值（通常轉化為一元函數問題）。
        3. 比較上述所有點的函數值，最大者為絕對最大，最小者為絕對最小。
*   **練習題 6.5.1**：
    求 $f(x, y) = x^2 - 2xy + 2y$ 在矩形區域 $D = \{ (x, y) | 0 \leq x \leq 3, 0 \leq y \leq 2 \}$ 上的絕對極值。
    *   **解答**：
        1. **內部**：$f_x = 2x - 2y = 0, f_y = -2x + 2 = 0 \implies x = 1, y = 1$。
           $f(1, 1) = 1 - 2 + 2 = 1$。
        2. **邊界**：
           - $y=0 (0 \leq x \leq 3): g(x) = x^2$。端點 $f(0, 0)=0, f(3, 0)=9$。
           - $y=2 (0 \leq x \leq 3): g(x) = x^2 - 4x + 4 = (x-2)^2$。端點 $f(0, 2)=4, f(3, 2)=1$，內部極值 $f(2, 2)=0$。
           - $x=0 (0 \leq y \leq 2): g(y) = 2y$。端點 $f(0, 0)=0, f(0, 2)=4$。
           - $x=3 (0 \leq y \leq 2): g(y) = 9 - 6y + 2y = 9 - 4y$。端點 $f(3, 0)=9, f(3, 2)=1$。
        3. **比較**：值有 $\{1, 0, 9, 4, 1, 0, 4, 9, 1\}$。
           絕對最大值為 9，發生在 $(3, 0)$；絕對最小值為 0，發生在 $(0, 0)$ 與 $(2, 2)$。

## 二、 動手實作 (Lab) - 總計 50 分鐘

### 實作：使用 SymPy 找尋臨界點並分類
**任務目標**：自動化尋找複雜函數的極值點並進行二階導數測試。

```python
import sympy as sp

def analyze_extrema(f_expr, vars_list):
    x, y = vars_list
    # 1. 找臨界點
    fx = sp.diff(f_expr, x)
    fy = sp.diff(f_expr, y)
    critical_points = sp.solve([fx, fy], (x, y), dict=True)
    
    # 2. 計算二階導與 D
    fxx = sp.diff(fx, x)
    fyy = sp.diff(fy, y)
    fxy = sp.diff(fx, y)
    D = fxx * fyy - fxy**2
    
    print(f"分析函數: {f_expr}")
    for p in critical_points:
        # 代入點值 (轉為 float 以方便判斷)
        try:
            D_val = float(D.subs(p))
            fxx_val = float(fxx.subs(p))
            f_val = float(f_expr.subs(p))
            
            result = ""
            if D_val > 0:
                result = "局部極小值" if fxx_val > 0 else "局部極大值"
            elif D_val < 0:
                result = "鞍點"
            else:
                result = "測試無結論"
                
            print(f"點 {p}: f={f_val:.4f}, D={D_val:.4f}, 結果: {result}")
        except:
            print(f"點 {p}: 包含複數或無法計算")

# 測試
x, y = sp.symbols('x y', real=True)
f = x*y * sp.exp(-x**2 - y**2)
analyze_extrema(f, [x, y])
```
