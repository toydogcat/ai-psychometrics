本文件提供了第 3 週「向量函數與空間曲線」課後測驗題庫共 30 題的完整解答與詳細剖析。

## 一、 單選題 (Single Choice) 詳細解析

1. **Q1**: 向量函數 $\mathbf{r}(t) = \langle \cos t, \sin t, t \rangle$ 的圖形名稱是？
   - **解析**: $x^2 + y^2 = 1$ 代表投影在 $xy$ 平面是圓，而 $z$ 隨時間增加，形成螺旋上升的曲線。
   - **答案**: (C)

2. **Q2**: 若 $\mathbf{r}(t) = \langle t, t^2, t^3 \rangle$，則 $\mathbf{r}'(t) =$？
   - **解析**: 分量求導：$(t)' = 1, (t^2)' = 2t, (t^3)' = 3t^2$。
   - **答案**: (A)

3. **Q3**: 單位切向量 $\mathbf{T}(t)$ 的長度恆為？
   - **解析**: 單位向量的定義即為長度等於 1。
   - **答案**: (B)

4. **Q4**: 直線的曲率 $\kappa$ 為？
   - **解析**: 直線不彎曲，切向量方向不變，故曲率為 0。
   - **答案**: (B)

5. **Q5**: 弧長參數 $s(t)$ 的微分 $ds/dt$ 等於？
   - **解析**: $s(t) = \int |\mathbf{r}'(u)| du$，根據微積分基本定理，$ds/dt = |\mathbf{r}'(t)|$。
   - **答案**: (B)

6. **Q6**: 若 $\mathbf{r}(t) = \langle 3t, 4t, 0 \rangle$，其弧長積分中的速度 $|\mathbf{r}'(t)|$ 為？
   - **解析**: $\mathbf{r}'(t) = \langle 3, 4, 0 \rangle$，大小為 $\sqrt{3^2+4^2+0} = 5$。
   - **答案**: (C)

7. **Q7**: 單位法向量 $\mathbf{N}(t)$ 的方向與 $\mathbf{T}'(t)$ 的關係是？
   - **解析**: $\mathbf{N}(t) = \mathbf{T}'(t) / |\mathbf{T}'(t)|$，故兩者方向相同。
   - **答案**: (C)

8. **Q8**: 半徑為 5 的圓，其曲率為？
   - **解析**: 圓的曲率 $\kappa = 1/R = 1/5 = 0.2$。
   - **答案**: (C)

9. **Q9**: $\int \langle t, 1, 0 \rangle dt$ 的結果包含？
   - **解析**: 分量積分：$\int t dt = t^2/2 + C$。
   - **答案**: (A)

10. **Q10**: 空間曲線 $\mathbf{r}(t)$ 為光滑曲線 (Smooth) 的條件是 $\mathbf{r}'(t)$：
    - **解析**: 光滑定義為 $\mathbf{r}'(t)$ 連續且在區間內不為 $\mathbf{0}$。
    - **答案**: (B)

## 二、 多選題 (Multiple Choice) 詳細解析

11. **Q11**: 關於向量函數導數的性質，哪些正確？
    - **解析**: (A, B, C) 是正確的乘積法則。(D) 錯誤，例如 $\mathbf{u}(t) = \langle \cos t, \sin t \rangle$，$|\mathbf{u}| = 1$，導數為 0，但 $|\mathbf{u}'| = 1$。
    - **答案**: (A, B, C)

12. **Q12**: 下列哪些向量在曲線任一點處互相垂直？
    - **解析**: Frenet 基底 $\{\mathbf{T, N, B}\}$ 構成右手正交基底，兩兩相互垂直。
    - **答案**: (A, C, D)

13. **Q13**: 曲率 $\kappa$ 可以表示為：
    - **解析**: (A, B, C) 皆為曲率的定義或推導公式。(D) $\rho$ 是曲率半徑，定義即為 $1/\kappa$。
    - **答案**: (A, B, C, D)

14. **Q14**: 若一質點沿曲線運動，其路徑長度與哪些因素有關？
    - **解析**: (A) 積分區間。(B) 速度大小（速率）。(C) 起點不影響長度計算（只要區間固定）。(D) 加速度不直接出現在長度公式中。
    - **答案**: (A, B)

15. **Q15**: 向量函數 $\mathbf{r}(t) = \langle t, \sin t, 2 \rangle$ 在 $z=2$ 平面上。這意味著：
    - **解析**: (A, B) 它是平面曲線，其 $z$ 分量為常數，導數為 0。
    - **答案**: (A, B, C)

16. **Q16**: 螺旋線 $\mathbf{r}(t) = \langle a\cos t, a\sin t, bt \rangle$ 的性質包括：
    - **解析**: (A, B, C) 皆正確。速率為 $\sqrt{a^2+b^2}$，曲率為 $a/(a^2+b^2)$，皆為常數。
    - **答案**: (A, B, C)

17. **Q17**: 關於弧長參數 $s$，下列敘述正確的是：
    - **解析**: (A) $|d\mathbf{r}/ds| = |\mathbf{v}| / |ds/dt| = |\mathbf{v}|/|\mathbf{v}| = 1$。(B, D) 皆為弧長參數化的性質。
    - **答案**: (A, B, D)

18. **Q18**: 若 $\mathbf{r}'(t) \cdot \mathbf{r}''(t) = 0$，則：
    - **解析**: 速度垂直於加速度。由 $\frac{d}{dt}|\mathbf{v}|^2 = 2\mathbf{v} \cdot \mathbf{a} = 0$ 可知，速度大小（速率）為常數。這也意味著切線加速度為 0。
    - **答案**: (A, B, C)

19. **Q19**: 下列哪些函數代表同一條空間曲線（僅參數化不同）？
    - **解析**: (A) 軌跡為 $(x, x^2, x^3)$。(B) 令 $t = e^u$，則軌跡亦為 $(t, t^2, t^3)$。 (C, D) 的軌跡與 A 不同。
    - **答案**: (A, B)

20. **Q20**: 求空間曲線交點時，正確的步驟包括：
    - **解析**: (A, B, C) 是標準求法。必須使用不同參數名，因為兩曲線可能在不同「時間」到達同一點。
    - **答案**: (A, B, C)

## 三、 填充題 (Fill-in-the-blank) 詳細解析

21. **Q21**: 若 $\mathbf{r}(t) = \langle 2t, t^2, \ln t \rangle$，則 $\mathbf{r}(1) = \langle 2, 1, $ __________ $\rangle$。
    - **解析**: $\ln 1 = 0$。
    - **答案**: 0

22. **Q23**: 圓 $x^2 + y^2 = 1$ 的曲率 $\kappa = $ __________。
    - **解析**: 半徑 $R=1$，曲率 $\kappa = 1/1 = 1$。
    - **答案**: 1

23. **Q23**: 向量函數 $\mathbf{r}(t)$ 的二階導數 $\mathbf{r}''(t)$ 在物理上代表 __________。
    - **解析**: 位置的二階導數是加速度。
    - **答案**: 加速度 (或 Acceleration)

24. **Q24**: 計算 $\int_0^1 \langle t, t^2, t^3 \rangle dt = \langle 1/2, 1/3, $ __________ $\rangle$。
    - **解析**: $\int_0^1 t^3 dt = [t^4/4]_0^1 = 1/4$。
    - **答案**: 1/4

25. **Q25**: 單位切向量 $\mathbf{T}(t)$ 的微分 $\mathbf{T}'(t)$ 始終與 $\mathbf{T}(t)$ __________。
    - **解析**: 單位向量與其導數必垂直。
    - **答案**: 垂直 (或 Orthogonal)

26. **Q26**: 弧長 $L = \int_a^b \sqrt{ (x')^2 + (y')^2 + (z')^2 } dt$。其中 $x', y', z'$ 是對 __________ 的導數。
    - **解析**: 對參數 $t$ 求導。
    - **答案**: 參數 t (或 t)

27. **Q27**: 曲率半徑 $\rho$ 與曲率 $\kappa$ 的關係為 $\rho = $ __________。
    - **解析**: 互為倒數。
    - **答案**: 1/κ

28. **Q28**: 螺旋線 $\mathbf{r}(t) = \langle \cos t, \sin t, t \rangle$ 的速度大小 $|\mathbf{r}'(t)| = $ __________。
    - **解析**: $\sqrt{(-\sin t)^2 + \cos^2 t + 1} = \sqrt{2}$。
    - **答案**: $\sqrt{2}$

29. **Q29**: 若曲線的曲率恆為 0，則該曲線必為 __________。
    - **解析**: 曲率為 0 代表方向完全不變。
    - **答案**: 直線

30. **Q30**: 單位法向量 $\mathbf{N}$ 的長度為 __________。
    - **解析**: 既然是「單位」法向量，長度必為 1。
    - **答案**: 1
